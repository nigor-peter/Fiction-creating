# 青梧探险写作

## Overview

Use this skill to write inside the original 《雾岭灯火》 world: a 1998 Chinese county-town youth adventure about 雾岭, 地下青梧, 引魂灯, memory, loss, and folk suspense.

Keep the work original. Borrow only broad, non-copyrightable craft patterns such as ensemble investigation, dual timelines, chapter-end hooks, small-town atmosphere, and emotional stakes. Do not reproduce specific characters, monsters, scenes, institutions, dialogue, or signature set pieces from any existing film or series.

## Workflow

1. Identify the requested output: character, scene, chapter, outline, mystery logic, folklore detail, revision, or continuity check.
2. Load only the needed reference:
   - `references/series-bible.md` for core world rules and story canon.
   - `references/writing-patterns.md` for prose, suspense, chapter structure, and originality guardrails.
3. Preserve established facts unless the user explicitly asks for a reboot or alternate version.
4. Center each output on concrete Chinese county-town details, active investigation, and character desire.
5. End creative outputs with a useful next hook when appropriate: an image, object, line of dialogue, discovery, or contradiction.

## Output Rules

Write in Chinese unless the user asks otherwise.

Prefer vivid, usable story material over abstract explanation.

Use named places, props, sensory details, and character choices.

Make supernatural rules emotionally costly. The danger of 地下青梧 is not only attack, but temptation: it can offer the return of someone lost.

For outlines, include episode/chapter function and ending hook.

For scenes, include action, atmosphere, subtext, and a visible turn in the final beat.

For continuity checks, list contradictions first, then offer minimal fixes that preserve the strongest drama.
