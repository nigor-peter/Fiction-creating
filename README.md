# 雾岭灯火 Fiction Creating

《雾岭灯火》是一个原创长篇小说创作项目，类型定位为 **少年探险 / 中国民俗悬疑 / 超自然冒险**。

故事发生在 1998 年夏天的西南山区青梧县。四个少年因同伴周尧失踪，追进一座不该存在的地下县城，逐渐发现“引魂灯”、旧县志、灯戏唱本、地磁实验与成人旧案之间的联系。

## 核心卖点

**中国县城的真实质感**  
旧电影院、录像厅、供销社、县医院、防空洞、水库旧城、山神庙与灯戏文化构成故事的日常底色。

**少年群像推动探险**  
林照野、沈星河、马小满、顾南舟各自拥有线索入口，也各自背负一个无法轻易说出口的失去。

**民俗悬疑有明确规则**  
地下青梧不能凭空创造完整的人，只能模仿被强烈记住的人。照片、录音、皮影和镜面会留下两个世界重叠时的痕迹。

**长篇引擎可持续**  
每章围绕一个现实问题、一个超自然异常、一次人物关系推进和一个具体悬念物展开，便于继续扩写、改编或拆分成剧集结构。

## 四卷概览

**第一卷：雾岭灯火**  
同伴失踪，旧电影院与地下青梧出现。核心问题是：失去的人是否真的能回来？

**第二卷：沉城旧名**  
水库旧城与名字规则展开。核心问题是：一个人被抹去名字后，是否还存在于世界上？

**第三卷：北斗渡**  
七座渡口与未走之路浮现。核心问题是：如果能改写过去，谁有权决定哪条路成为现实？

**第四卷：无声林**  
地图空白、学校失声、林场旧证与第零生产队揭开最终旧债。核心问题是：谁有资格被记住？大结局回收“回来、名字、方向”三条命题，让青梧县以共同记忆而非个人牺牲完成收束。

## 主要人物

- **林照野**：照相馆老板的儿子，旧相机持有者，负责记录与追问真相。
- **沈星河**：转学生，擅长无线电和旧电器，是成人旧案的重要入口。
- **马小满**：小卖部孩子，消息灵通，代表普通人的勇气与具体代价。
- **顾南舟**：县剧团退休武生的孙子，行动担当，背后牵连哥哥失踪线。
- **阿铃**：来自异常世界的神秘女孩，连接灯戏、引魂灯与边界规则。
- **周尧**：失踪同学，触发整个故事，也是“真假归来”命题的核心。

## 写作原则

本项目保持原创。可以借鉴少年群像、怀旧县城、双线调查、现实危机与超自然并行等通用叙事方法，但不复刻任何现成作品的人物、场景、怪物、对白或标志性桥段。

创作时优先遵循：

- 悬疑必须可追查，不只靠气氛堆叠。
- 民俗元素必须服务人物选择和故事规则。
- 每个超自然异常都要落回一个具体的人情代价。
- 少年线负责行动与情感推进，成人线负责旧案与历史债务。
