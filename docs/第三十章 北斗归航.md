# 第三十章 北斗归航

北斗渡第一次由所有乘客共同掌舵。

沈星河不再握住司南灯。她把七束火分别引向七枚星钉，每个渡口只回答自己的方向。

顾南舟走边稳住船身。

马小满站在船舱中央，大声念出每个人的名字，防止路线重叠时彼此遗忘。

周尧守住青梧航标。

江晚和江潮一起守白鹭镇航标。姐姐不再把弟弟拉回岸，弟弟也答应每年在山洪纪念日点三下灯，告诉家里自己仍在选择。

林照野没有拍航行过程。

他只在每一条路线稳定后，问那里的乘客愿不愿意留下记录。有人点头，他才按快门；有人摇头，他便放下相机。

阿铃站在七星中央。

司南灯不再嵌进她手掌。七束火照着她，却没有把她拉向任何一个时代。

“你想去哪？”林照野问。

阿铃想了很久。

从前她总知道别人该走哪条灯路，却从没人问过她自己的方向。

“我想看看开学。”她说。

马小满笑出声：“这个愿望很危险。开学有三本暑假作业追着你。”

阿铃认真问：“作业会吃人吗？”

“比归人厉害。”

陆沉舟的小船靠近无向船。

他的影子已经回来，左手缺失的小指也留下真实疤痕。他不再是北斗渡唯一摆渡人，只是陆家渡航标的守护者。

“我能下船了。”他说。

岸上没有女儿，也没有恢复的村庄。

只有新立的村口碑：陆家渡，曾居一百八十六人。

他伸手摸碑，像终于摸到自己曾经存在过的证据。

“够吗？”马小满问。

陆沉舟摇头：“不够。”

停了一下，他又说：“但是真的。”

七枚星钉固定完成。

司南灯的铜壳裂成七片，分别飞向七座渡口。北斗渡仍然存在，却不再听从一个人的愿望。此后船要靠岸，必须同时满足两件事：乘客选择过去，岸上的人愿意回应。

没有回应的岸，不能被强行闯入。

无向船回到青梧码头时，现实只过去一夜。

开学第二天清晨，五个孩子从废弃候船室走出来。衣服全湿，书包里装满船票、星图和水草。

沈岚、林成川、马莲和顾怀声都在码头等。

没人先问发生了什么。

沈岚先抱住女儿。

林成川看见儿子没有举相机，也没有催他拍照。

马莲一边哭一边骂马小满刚开学就逃课。

顾怀声接过顾南舟的木刀，看见刀柄上多了一道顾北川留下的刻痕。

周尧妈妈叫出儿子的名字，没有迟疑。

江晚没有跟他们上岸。

她要先回白鹭镇。江潮站在第三渡口，航标在晨雾里闪了三次。

阿铃最后一个走出黑门。

她穿着青布褂，脚上却是一双马小满从小卖部拿来的塑料凉鞋。胸前姓名牌写着：阿铃，青梧县小学，年级待定。

黑门缓缓关闭。

门上七颗星变成七个铜钉。

沈星河拔下其中一枚。

星钉在她掌心里轻轻指向水库，又慢慢停住。

“还能打开吗？”林照野问。

“可以。”

“能改过去吗？”

“不能。”沈星河把星钉收好，“只能在真正有人选择来、也有人愿意接的时候找路。”

新学期地理课上，老师重新挂起中国地图。

所有指南针恢复指北。

阿铃坐在最后一排，第一次拿铅笔写自己的名字。她写得很慢，每一笔都像在确认自己真的坐在这里。

林照野抬头看地图。

西北角多出一片灰绿色区域。

没有省界，没有地名，也没有公路河流。地图图例里写着：未勘测林区。

只有林照野他们看见，那片林区上方笼着一圈不自然的白。

不是雾。

像声音被擦掉后留下的空白。

放学铃响后，阿铃走到窗边。

她用手指在玻璃上写：

无声林。

“那里有什么？”顾南舟问。

阿铃摇头。

“没有灯。”

“没有灯不是好事吗？”马小满说。

“也没有名字。”沈星河看着地图。

阿铃补上最后一句：

“那里的人，从来没有走失过。”

林照野觉得这句话比任何失踪都更冷。

因为只有从未被寻找的人，才不会被称为走失。

窗外传来一声很远的伐木声。

咚。

灰绿色林区在地图上扩大了一小块。
