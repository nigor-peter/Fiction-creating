# 第十七章 沉城戏台

顾南舟是在爷爷的戏箱里找到那张票的。

戏箱压在床底，外面包着油布，里面放着顾怀声年轻时的靠旗、旧靴、髯口和一把没有开刃的木刀。最底下夹着半张青梧影剧院戏票。

票是顾北川留下的。

正面写着晚场七点，背面不是文字，而是一串小人图。每个小人姿势不同，有的抬腿，有的转身，有的单脚踩在方格上。

顾怀声坐在门口抽烟，看了很久。

“这是武生走边。”他说。

“什么？”

“戏台上走夜路的步法。前无灯，后有追兵，脚踩错一步，就算跌下悬崖。”

顾南舟把票摊开：“能走到哪里？”

老人拿烟杆点住最后一个姿势。

“台心。”

他抬头看孙子。

“你哥画这个，不是教你进去，是教你出来。”

第二个林照野正在长大。

不是身体长大，而是越来越真实。

照相馆的镜子里偶尔会出现他；暗房红灯闪一下，墙上就多出两个人影；林照野写作业时，练习册上会多出另一种笔迹，替他写完下一道题。

最让人不安的是，那笔迹比他自己的更像林成川。

“它从你的名字里学你。”沈星河说。

她在桌上画出两盏灯和两条线。

“第一盏灯用记忆造影子，第二盏灯给影子名字。现在那个东西既有你的记忆，又占了你的名字位置。”

马小满问：“怎么分？”

沈星河沉默。

林照野知道答案。

同名者，只能留一个。

可“留”不一定意味着活下来，也可能意味着谁更被别人相信。

黄昏前，他们在水库堤坝集合。

沈岚和林成川负责留在岸上，配合水库管理所疏散附近居民。马莲拿着旧搬迁名册，帮助马小满核对万灯巷姓名。顾怀声把武生靴和木刀交给顾南舟，只说了一句话：

“走边不能回头。”

阿铃带来一张沉城图。

图上有两条路线。

林照野、沈星河、马小满从万灯巷进入旧戏台地下，寻找第二盏灯的灯芯和点名册。

顾南舟与阿铃从台面走边，阻止赵砚秋完成最后唱名。

周尧被留在岸上。

他反对，但所有人都知道校服周尧仍借着他的名字存在。带他下去，等于给影子开门。

修堤船入水时，雨又开始下。

这次没有老吴撑船。

阿铃站在船头，用竹篙一点水面。旧青梧便从水下浮起，像一张等待显影的巨大相纸。

五个人在万灯巷口分开。

顾南舟换上武生靴，把木刀别在腰后。阿铃手里提着纸灯笼。两人踏上旧戏台外沿时，台下所有空椅都转向他们。

顾南舟按照戏票走步。

左三，右一，转身，亮相。

每一步踩下，腐朽木板下都会亮起一盏灯。走错的地方则是一片看不见底的黑水。

阿铃跟在他身后唱：

“前无灯，后无门。”

“一步错，旧名沉。”

走到第七步时，顾北川出现在戏台中央。

这次他没有劝顾南舟回家，也没有叫弟。

他穿着三年前的黑汗衫，坐在台边，手里把玩另一半戏票。

“你终于看懂了。”他说。

顾南舟停在方格外。

“这是你留下的？”

顾北川点头。

“赵砚秋会从台心叫名。走边能避开灯的位置。”

“你是真的？”

顾北川笑了一下。

“你还在问这个？”

顾南舟握紧木刀。

顾北川把半张戏票递出来。两半若能拼上，就证明至少这段记忆是真的。

“让我回来。”他说，“我能带你们到赵砚秋那里。”

阿铃在后面轻声说：“不能接。”

顾北川看向她：“你送了那么多人，凭什么不送我？”

阿铃回答：“送路，不是送回原处。”

顾北川的眼神暗下去。

“南舟，我在这里三年了。”

顾南舟眼里全是血丝。

“我知道。”

“你知道有多冷吗？”

“不知道。”

“那就拉我一把。”

顾南舟伸出手。

阿铃没有拦。

他的手停在半空，离半张戏票只有一寸。

然后，他从腰后抽出木刀，劈断了两人之间亮起的灯线。

“我来不是带你回去。”顾南舟说，“我是来把你送完。”

顾北川脸上的痛苦停了一下。

很轻地，他笑了。

像哥哥终于看见弟弟学会了自己留下的最后一招。

另一边，林照野三人进入戏台地下。

地下不是地窖，而是一座倒悬的观众厅。座椅钉在头顶，舞台在脚下，所有姓名牌悬在空中。沈星河用收音机寻找灯的低频，马小满抱着纸灯笼，逐一抄下无名者残缺的名字。

林照野负责拍路。

肉眼看不见的门，会在照片里出现。

他们找到点名册时，另一个林照野也来了。

他从暗房一样的红光里走出，有脸、有影子、有相机。

“你拍得太慢。”他说。

林照野看着他。

“你是谁？”

“我是你没拍下来的那些部分。”

另一个林照野举起相机。

“爸更喜欢我。我不会害怕，也不会错过真相。”

闪光灯亮起。

林照野眼前一白。

再睁眼时，沈星河和马小满已经不见了。

他站在照相馆暗房里，林成川正笑着对他说：

“照野，这次你拍对了。”

可林照野知道，父亲已经很多年没有这样笑。

他把自己的相机放到地上。

“你想要名字，拿去。”

另一个林照野愣了。

“但相机不给你。”

林照野抬脚踩碎地上的镜面。

暗房裂开。

他回到戏台地下，沈星河正拉着他的手，马小满用点名册拍他脸。

“醒醒！另一个你比你会写作业，但说话更讨厌！”

林照野喘着气站起来。

点名册就在他们脚边。

第一页写着赵砚萤。

第二页写着沈怀川。

第三页写着顾北川。

第四页，墨迹正在组成：林照野。

头顶戏台传来赵砚秋的声音。

“第一位归人，赵砚萤。”
