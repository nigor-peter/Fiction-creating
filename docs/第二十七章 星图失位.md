# 第二十七章 星图失位

青梧县小学多了一百三十二名学生。

他们穿白鹭镇蓝白校服，坐在教室和走廊里，坚称自己一直在这里上学。青梧老师认识其中一半人的名字，白鹭镇老师却出现在县医院，认定那里是镇政府。

两个现实正在互相覆盖。

沈岚在医院看见另一个自己。

那个沈岚没有嫁给沈怀川，也没有女儿。她是白鹭镇防疫站医生，负责山洪后的救治。两个人站在同一条走廊，拥有部分相同记忆，却过着完全不同的人生。

周尧的情况最糟。

未淹青梧路线侵入后，学校花名册再次没有他的名字。同学看见他会觉得眼熟，却想不起在哪见过。连周尧妈妈叫他时，也会短暂迟疑。

“我又要被删一次？”周尧问。

林照野说：“不会。”

“你拿什么保证？”

林照野答不出来。

无向船在错位星图下剧烈摇晃。每个人脚下都同时出现几个世界：青梧老街、白鹭镇码头、未淹旧城、灯市街和北斗渡。

司南灯悬在桅杆上，七道火光彼此拉扯。

陆沉舟说：“必须有人接灯。”

沈星河看着自己的手。

只要握住灯，她就能恢复星位，送所有人回到原来的世界。代价是自己永远留在船上。

马小满挡在她前面：“不行。”

“如果不拿，两个县都会乱。”江晚说。

“所以就让她一个人没家？”

“总得有人承担。”

“那为什么不是你？”顾南舟问。

江晚脸色一沉：“是我改错了路，我可以拿。”

她伸手，司南灯却转开。

灯不选她。

沈星河说：“我拿。”

林照野抓住她：“再想。”

“现在每拖一分钟，现实里就有人忘记自己。”

“那也不能把你算成代价。”

“我不是被算。我自己选。”

团队第一次真正吵起来。

沈星河认为个人有权牺牲自己；林照野认为旧规则正利用她的责任感；江晚坚持先稳定多数人；马小满要求先说出沈星河将失去什么；顾南舟则拔刀准备砍断灯柄。

周尧忽然大喊：“都闭嘴！”

没人想到最先失去名字的人会叫停他们。

“你们又在讨论谁可以不存在。”他说，“第一卷是假的我，第二卷是没名字的人，现在轮到沈星河。每次都说没办法，每次都是先挑一个人扔下去。”

船上安静下来。

周尧看向沈星河：“你想救我们，可以。但我们也有权不接受这种救法。”

沈星河眼圈红了。

陆沉舟站在船尾，看着他们。

林照野突然发现，他脚下没有影子。

不只是没有灯影。

水面、甲板、星光，任何地方都映不出陆沉舟。

“你不是摆渡人。”林照野说。

陆沉舟看向他。

“你也是被留下的路线。”

相机快门自行按下。

照片里没有陆沉舟，只有一座被洪水抹平的村庄。村口碑上写着：陆家渡，全村四十九户。

照片背面出现日期，一九五四年。

江晚拿起照片：“这个地方不存在。”

“存在过。”陆沉舟说。

他终于摘下斗笠。

“我用司南灯救了女儿。改路以后，她活了，陆家渡却替她沉了。现实里没有村庄，也没有我。我留在渡河，不是因为摆渡人不能选岸。”

他看向司南灯。

“是因为我已经没有岸。”

第七颗星完全熄灭。

无向船向黑水深处倾斜。

陆沉舟说：“想活着回去，就在七星全灭前，决定哪条现实应当留下。”
