# 第三十四章 不响的铃

陈守根把他们带到林场后面的晒木场。

晒木场荒了多年，地上还钉着几排铁轨，用来拖运原木。铁轨缝里长满野草，秋风一过，草叶贴着生锈的铁边，发出细细的摩擦声。

“再往里就是无声沟。”陈守根说，“以前不叫这个名。”

“叫什么？”周尧问。

陈守根看了他一眼：“说不出来。”

老人不是不肯说，而是真的说不出来。他张了几次嘴，喉结滚动，最后只吐出一口发干的气。

阿铃站在铁轨尽头，忽然抬起手。

晒木场边有一根木桩，木桩上挂着一只铜铃。铃身发黑，裂了一道细缝，里面没有铃舌。

阿铃走过去，手还没碰到，铃先晃了一下。

没有声音。

她脸色却变了。

“我听见了。”她说。

“听见什么？”林照野问。

“有人叫孩子回家。”

顾南舟皱眉：“这铃都没舌头，怎么响？”

陈守根低声说：“那不是给活人听的。”

阿铃把铜铃取下来。铃很轻，轻得不像铜，倒像一块被火烧空的木头。她刚握住，晒木场四周的树影忽然往前挪了一寸。

马小满立刻躲到顾南舟身后。

顾南舟举起木棍：“谁？”

没有人。

只有树影继续往前。

它们不是被太阳拉长，而是像一群黑色的人，贴着地面走过来。每一根影子都没有头，只有拖长的肩膀和手。

顾南舟突然把木棍往地上一敲。

这是他爷爷教他的戏台开场步，武生上场前先定场。木棍落下，没有响声。

树影停了一下。

顾南舟眼睛一亮，又敲第二下、第三下。他敲不出声，但他的脚步有节奏，肩背绷直，像旧戏台上一个看不见锣鼓的将军。

树影退了一步。

“有用！”马小满无声地喊，喊完才发现自己又没声音了。

阿铃胸前空白姓名牌忽然裂开。裂缝里透出很弱的黄光，像一粒快灭的火星。

她闭上眼。

林照野看见她身后站着一个小女孩。

不是完整的人影，只是一截从旧照片里走出来的轮廓。女孩胸前也挂着铜铃，脸仍旧空白。

“她是谁？”林照野问。

阿铃轻声说：“我不知道。”

她停了停。

“可我好像一直在替她响。”

陈守根背过身去，肩膀塌得厉害：“第零生产队有个孩子，采灯芯木那年才十一岁。她负责在雾里摇铃，工人听见铃就知道往哪边走。”

“后来呢？”沈星河问。

“后来灯点起来了。”陈守根说，“人没回来。”

晒木场尽头的树影忽然分开。

一个孩子站在那里。

他看上去十三四岁，头发剪得很短，身上穿着旧式蓝布衣，脚上是一双草鞋。他的眼睛黑得像没有光照进去过。

“你们找失踪的人。”他说。

他的声音很轻，却所有人都听见了。

“我们找从没被找过的人。”

周尧往前一步：“你是谁？”

孩子看着他，像在看一个很稀奇的东西。

“你被找回来了。”他说，“所以你有名字。”

周尧脸色一白。

“那你呢？”

孩子低头看了看自己胸口。那里没有姓名牌，也没有别针孔。

“他们有时候叫我何栖。”

“有时候？”

何栖抬头，指向林场深处。

“只要有人记得，才算名字。没人记得的时候，那只是声音。”

他转身往林子里走。

阿铃手里的铜铃无声地晃了三下。

树影像水一样合拢。

何栖的声音从林子里传出来：

“想知道她是谁，就把所有被借走的灯还回来。”
