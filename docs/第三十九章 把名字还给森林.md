# 第三十九章 把名字还给森林

第一个站出来的是马莲。

她把小卖部赊账本摊开，手指按在一页几乎看不见的压痕上。

“何栖。”她说。

声音很轻，却没有消失。

“买过桂花糖，两块。欠五分钱。”

人群里有人笑了一下，笑声刚出来就哽住了。

何栖站在广场边缘，眼睛慢慢红了。

第二个是陈守根。

他背着柴刀走到证据堆前，手抖得厉害。他把一张伐木证放下，说：“陈月苗，十七岁，爱唱山歌，唱得不好，嗓门大。七月十五那天，她负责给采木的人煮饭。”

广场上的一盏路灯亮了。

不是借来的亮。

那光很稳，像终于知道自己该照向谁。

接着，顾怀声走上来，放下一本旧戏谱。

“罗大顺，会打小锣，老是抢拍。”

沈岚放下一张旧病历。

“岑文秀，右手虎口被锯伤，缝了四针，没喊疼。”

赵砚秋放下县志补遗。

他声音发哑：“青梧县第零生产队，共三十一人。因灯芯木采伐与林场事故长期未登记，未补偿，未立碑，未入志。”

他说完，向广场深处鞠了一躬。

无声林的树影从街口漫进来。

没有风，树影却像水一样流过石板路，停在证据堆前。

阿铃走到最前面。

她胸前两个名字还在争。岑小铃亮一下，阿铃亮一下，像一盏灯在两个时代之间来回晃。

何栖看着她：“你可以回来。”

“回哪里？”

“回我们那里。”何栖说，“你是最后的铃。”

阿铃低头看自己的塑料凉鞋。那是马小满从小卖部拿的，鞋面上还印着一只褪色的小花。

她又看向林照野、沈星河、顾南舟、周尧。

最后看向马小满。

马小满别过脸：“你别看我，我肯定说不出什么大话。”

阿铃笑了一下。

“那你说小话。”

马小满吸了吸鼻子：“阿铃，欠我一双凉鞋钱。还没还。”

广场上有人笑了。

这一次笑声没有被吞掉。

阿铃胸前的姓名牌停止闪烁。

“岑小铃是旧名字。”她说，“我要把它还给林子。那里有人等这个铃声回去。”

她把手按在姓名牌上。

岑小铃三个字从白塑料上浮起，像一小片温热的光，慢慢飘向树影。

“阿铃是现在的名字。”她继续说，“是你们叫出来的。我留下。”

树影没有愤怒。

它们只是安静地接住那三个旧字。

无声林停止扩张。

地图上的灰绿色不再往县城边缘爬，广场上的声音一点点回来。有人哭，有人喊，有人开始一个接一个说名字。

每喊一个名字，就必须说一件具体的事。

不能只说他们牺牲。

要说他们买过糖、修过路、唱跑过调、欠过钱、怕过黑、喜欢过谁、在雨天借过伞。

因为只有这样，名字才不是碑上的墨。

它才像一个人。

夜深时，何栖走到林照野面前。

“还有最后一个问题。”

林照野看着他。

何栖问：“如果没有人记得你，你还愿意记得别人吗？”

广场上所有声音都低下去。

这个问题不是问林照野一个人。

是问整个青梧。

周尧走出来。

他曾经差一点变成一个无法证明的人，所以他最先回答。

“愿意。”他说，“因为有人找过我。”

阿铃接着说：“愿意。因为有人给我现在的名字。”

沈星河说：“愿意。因为告别也要有人听见。”

马小满小声说：“愿意。欠账也得记。”

顾南舟把木棍往地上一顿。

没有戏台锣鼓，但节拍稳稳落下。

广场上的人开始一个接一个回答。

愿意。

愿意。

愿意。

无声林的树影终于往后退去。
