# 第七章 归人无声

七月十五不是后天。

这是青梧县在第二天清晨醒来后，所有人都意识到的事。

因为街口电线杆上的通知变了。

昨晚县广播明明说的是“七月十五日晚七点至十二点全城停电”，可清晨林照野路过老街口时，看见新贴的白纸上写着：

今晚七点。

没有日期。

没有原因。

只有四个字，黑得像刚从墨水里捞出来。

今晚七点。

林照野站在电线杆前，后背一点点发冷。

卖凉粉的婆婆也在看那张通知。她眯着眼念了一遍，嘀咕：“昨儿不是说后天吗？电力局这帮人，越修越乱。”

林照野没有说话。

他伸手摸了摸通知纸边缘。

纸是湿的。

像刚从井里贴上来。

中午前，青梧县开始传出有人回来的消息。

第一个是老粮站后面的魏家。

魏家老二五年前去外地打工，路上出了车祸，人没了。可今天早上，有人在粮站巷口看见他穿着出门那天的蓝夹克，站在自家门口，手里提着一袋苹果。

魏家老太太当场哭晕过去。

第二个是县医院旧楼。

一个死了七年的护工阿姨，被人看见坐在走廊尽头织毛衣。她抬头对小护士笑，说：“别怕，我值夜班。”

第三个消息传到小卖部时，马小满手里的搪瓷缸掉在地上，摔掉了一块瓷。

有人看见他外婆了。

不是在灯市街。

是在青梧县真正的老街。

“不可能。”马小满说。

他说这话的时候，脸色白得很。

林照野、沈星河和顾南舟都在小卖部后院。桌上摊着从旧档案室拿出来的地图、观测记录、顾北川的借条，还有赵砚秋后来偷偷塞给他们的一张纸。

纸上写着一串名字。

不是手写的，是从旧档案里抄出来的。

青梧归人名录。

1988 年七月十五异常事件后，有短暂目击记录者，共十一人。

其中三个名字被墨水划掉，剩下的八个名字里，有魏家老二，有医院护工，还有马小满外婆。

最后一行，是新添上去的。

周尧。

“谁添的？”顾南舟问。

林照野摇头：“赵砚秋说他没写。”

马小满盯着外婆的名字，嘴唇发抖：“我外婆是前年走的，怎么会在 1988 年的名单上？”

沈星河轻声说：“可能不是死亡名单，是被灯市街记住的人。”

“我宁愿你别解释。”马小满说。

外面忽然传来马小满妈妈的声音：“小满！你外婆以前用的蒲扇呢？”

马小满整个人僵住。

他妈妈还在前头翻箱倒柜，声音带着哭腔：“我记得放柜子上的，怎么找不到了？”

马小满冲出去。

林照野他们跟到门口，只看见小卖部门帘外站着一个老太太。

蓝布衫，白头发，手里拿着蒲扇。

她站在阳光里，不像灯市街那样潮湿，也不像梦。路过的人都看见了她。有人停下，有人惊叫，还有人小声念佛。

马小满妈妈站在柜台后面，脸上的表情像被人一刀割开。

“妈？”

老太太笑了。

“小莲，家里怎么乱成这样？”

她的声音和马小满记忆里一模一样。

马小满站在门槛后面，一只手死死抓着门框。

老太太转头看见他，眼睛弯起来。

“满伢子，又长胖了。”

这句话一下子击中了他。

不是灯市街的“外婆”。

不是反着的红绳。

这个老太太左手腕上系着红绳，结也打对了。她知道小卖部柜台下面藏着零钱，知道马小满妈妈小名叫小莲，甚至知道马小满小时候把半罐话梅藏进米缸里，害得全家吃了三天酸米饭。

马小满妈妈哭着从柜台后绕出来。

“妈……”

马小满突然冲上去，挡在她面前。

“别认！”

整条街都安静了。

马小满妈妈愣住：“你疯了？”

马小满嘴唇发抖，却没有让开。

“她不是外婆。”

老太太脸上的笑慢慢淡下去。

“满伢子，”她说，“你在灯市街不认我，回家了也不认我？”

马小满像被人扇了一巴掌。

林照野立刻明白了。

她记得灯市街。

她知道他们进去过。

沈星河低声说：“退后。”

可已经晚了。

周围有人开始窃窃私语。马小满妈妈哭着骂他不懂事，伸手要推开他。那老太太只是站着，眼神越来越安静，安静得不像一个终于回家的老人。

“小莲，”她轻声说，“你不想妈吗？”

马小满妈妈的手停住。

这句话太狠。

林照野看见马小满的肩膀在抖。他明明怕得要命，却仍然站在门口，像一根快断的木桩。

顾南舟忽然走上前，把归人名录摔在柜台上。

“你是哪一年回来的？”

老太太看向他。

顾南舟继续问：“1988？还是今天？”

老太太脸上的皱纹像忽然深了一点。

“小孩子不要多问。”

“那你告诉我，”顾南舟从口袋里拿出哥哥的借条，“顾北川在哪？”

老太太没有回答。

街口风起。

老街两边的电灯，明明是白天，忽然一盏接一盏亮了。

老太太慢慢后退一步。

她的脚下没有影子。

马小满妈妈终于看见了。

她捂住嘴，眼泪还挂在脸上，却不敢再往前。

老太太叹了口气。

“你们这些孩子，”她说，“总爱把门关上。”

她转身往巷子里走。

林照野追了两步，看见巷子尽头站着一个人。

周尧。

白背心，蓝短裤，浑身湿透。

他看见林照野，嘴唇动了动：

“别让她认。”

下一秒，巷口另一边又走出一个周尧。

校服湿漉漉贴在身上，袖口少了一颗扣子。

他也看着林照野。

“带我回家。”

两个周尧隔着一条巷子站着。

马小满外婆的身影从他们中间穿过去，像一阵雾。

顾南舟冲过去。

两个周尧同时后退，退进巷子深处。

白背心周尧喊：“周尧妈妈不能认错！”

校服周尧喊：“我妈在找我！”

声音一样急，一样像真的。

林照野听见远处传来哭声。

周尧妈妈正从粮站方向跑来，手里抓着一张寻人启事，像抓着最后一根绳子。

她看见巷子里的两个孩子，整个人停住。

“尧尧？”

沈星河脸色变了：“拦住她！”

林照野和顾南舟同时冲过去。

可周尧妈妈已经往前走了一步。

两个周尧都看向她。

一个说：“妈，我冷。”

另一个说：“妈，我回来了。”

周尧妈妈的脸一点点崩开。

她伸出手。

就在这时，阿铃的声音从巷子上方响起。

“归人无声。”

林照野抬头。

阿铃站在巷口屋檐上，手里提着那只没点亮的灯笼。

她看着周尧妈妈。

“真正回来的，会先忘记怎么叫人。”

周尧妈妈的手停在半空。

白背心周尧低下头，像被这句话刺痛。

校服周尧却仍然看着她，轻轻说：“妈。”

巷子里的灯全亮了。

阿铃跳下来，落地没有声音。她走到林照野身边，把一张折好的纸塞进他手里。

“今晚七点以前，找到灯。”

“在哪里？”林照野问。

阿铃看向雾岭方向。

“灯不在井里。”

“那在哪？”

她轻声说：

“在愿意认下它的人心里。”

马小满几乎崩溃：“能不能说点我们听得懂的？”

阿铃看了他一眼：“山神庙。”

说完，她转身走进巷子。

两个周尧、马小满外婆、那些亮起的街灯，同时像被风吹散的纸灰一样淡下去。

周尧妈妈跪在地上，哭得没有声音。

林照野打开阿铃给他的纸。

纸上是一幅很简单的图。

雾岭山神庙。

庙后有一盏灯。

灯旁边写着一句话：

认影为亲，亲即为门。

远处县广播响起。

“通知，今晚七点，全县停电。”

这一次，播音员的声音不是女播音员。

是周尧。

“请各位归人，按时回家。”
