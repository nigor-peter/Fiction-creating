# 第二十六章 引水童

江潮记得自己被洪水卷走。

他也记得醒来时躺在一块门板上，四周全是黑水。许多失踪者抱着木头、树枝和屋顶漂在河上，没有人知道岸在哪。

一艘无向船经过。

陆沉舟问他，要不要回到山洪前。

江潮没有答应。

“我回去，别人就会被冲走吗？”

陆沉舟说：“可能。”

于是九岁的孩子留在第三渡口，给后来的迷路者点航标。他不知道这算活着还是死去，只知道每亮一盏灯，就有一条船不必撞上暗礁。

“你为什么不回家？”江晚站在渡口，声音发抖。

江潮说：“我在做事。”

“你才九岁。”

“九岁也能点灯。”

“我来接你。”

江晚伸手。

江潮后退一步。

“姐，你问我了吗？”

这句话让江晚僵住。

她一直在计算怎样救弟弟，怎样改动最少的人，怎样让更多人活下来。可她从没想过，江潮不是一道需要她解开的题。

“你想留在这里？”

“现在想。”

“以后呢？”

“以后再选。”

江晚哭了。

她没有再往前。

渡口另一边，阿铃提着司南灯出现。

她的身体被七种光分开。一半穿青布褂，一半穿现代校服；脸有时是十一岁，有时像更小的铃娘。每一条路线都在把她拉向不同年代。

林照野冲过去。

“阿铃！”

阿铃看着他，像花了很久才认出名字。

“林照野。”

她手里的司南灯形似旧罗盘，中央火苗指向七个方向。灯柄已经嵌进她掌心，无法放下。

“它在找新的持灯人。”她说。

“让陆沉舟拿。”马小满说。

陆沉舟站在船上，没有动。

阿铃摇头：“他没有现实，灯找不到他的归处。”

“江晚？”

“她想替所有人选路。”

江晚脸色一白，却没有反驳。

司南灯忽然转向沈星河。

七道火光同时指住她。

沈星河后退一步。

陆沉舟说：“灯选中了你。”

“为什么？”

“因为你曾亲手切断最想听见的声音。”

持有司南灯的人必须能放弃自己的岸，否则每一次指路都会暗中偏向最想去的地方。

沈星河握住灯柄。

火光从阿铃掌心转到她手上。

一瞬间，她看见七条人生。

一条里父亲活着。

一条里母亲从未参加实验。

一条里她没有转学来青梧，不认识林照野他们。

一条里她成为北斗渡永远的持灯人，可以看见所有道路，却再也不能走上任何一岸。

沈星河松开手。

司南灯没有掉落，悬在她面前。

“如果我拿了，会怎样？”

阿铃说：“你能送我们回去。”

“我呢？”

“没有持灯人自己的岸。”

马小满立刻说：“不要。”

顾南舟也说：“想别的办法。”

沈星河看向陆沉舟：“以前的持灯人都一个人承担？”

“方向只能有一个。”

“谁规定的？”

陆沉舟沉默。

这句话他们已经问过一次。

以前的人定下的规矩，不一定永远正确。

沈星河没有接灯。

“先带阿铃上船。”

他们用红绳把司南灯固定在桅杆，不让它嵌进任何一个人的手。七道火光在船上摇晃，北斗星位也跟着移动。

江晚最后看了一眼弟弟。

江潮把一枚小航标递给她。

“我想回去的时候，会点三下。”

江晚接过，没有拉他。

船离开第三渡口。

刚驶出百米，天空中的北斗七星突然乱了。

第一颗星坠向东方，第四颗星分成两个，第七颗完全熄灭。

陆沉舟第一次变了脸色。

“有人改过倒流河的选择。”

江晚低头。

她阻止弟弟去庙会的那条改路，正在反过来改写整个星图。

远处水面上，两座县城同时出现。

青梧县小学的操场上，站满了穿白鹭镇校服的孩子。
