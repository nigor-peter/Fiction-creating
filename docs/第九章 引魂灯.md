# 第九章 引魂灯

雾岭最高处原本没有牌楼。

青梧县的人都知道。雾岭只有山路、废井、旧水渠和半塌的山神庙。可此刻，牌楼就立在山脊上，黑瓦白墙，匾额上写着四个字：

地下青梧。

林照野举起相机。

取景框里，牌楼后不是山。

是一座县城。

反着的门牌、湿漉漉的青石板、灯油铺、纸扎铺、青梧照相，全都在。灯市街不再是一条缝，它像一整座城，压在现实青梧上方。

七点以后，归人越来越多。

他们从雾里走出来，有的站在山路边，有的沿着水渠往县城方向去。每个人都很安静，不哭不笑，像终于等到开门。

沈岚和林成川被赵砚秋留在山神庙口。

他们不是不想跟。

而是灯路不让他们再往前。林成川往前走一步，就有年轻的林照野站在雾里叫他爸；沈岚往前一步，就有沈怀川的声音从井底传来。成年人欠下的东西太多，雾岭总能找到他们回头的理由。

“你们去。”沈岚对沈星河说。

沈星河看着她。

沈岚把那只旧手电塞给她，手指冰凉。

“不要找你爸。”她说，“找灯。”

沈星河点头。

林成川把黑布交给林照野。

“看见引魂灯，盖住它。”他说，“不要吹。不能用水。灯火不能散，只能遮。”

“为什么？”

“散了，影子会到处都是。”

林照野把黑布塞进书包。

赵砚秋把唱本撕下来的复印页递给马小满：“如果阿铃唱不下去，你接。”

马小满瞪大眼：“我？”

“你记性最好。”

“我记性好是用来记小卖部谁欠钱的。”

赵砚秋没有笑：“你怕，但你不会丢下人。”

马小满张了张嘴，最后把纸接过去。

他们跟着阿铃往牌楼走。

两个周尧站在牌楼下。

白背心周尧看起来更虚弱，脸色青白，嘴唇发紫，像随时会倒下。校服周尧却比之前更像活人，甚至脸上有了血色，眼睛里也有了林照野熟悉的亮光。

周尧原本就是这样。

得意、嘴硬、怕被人看扁。

校服周尧说：“你们为什么不带我回家？”

顾南舟握紧拳头。

白背心周尧低声说：“别听他。”

校服周尧看向顾南舟：“你不想救我了？”

“想。”顾南舟说，“所以我要知道你是谁。”

“我是周尧。”

“你最怕什么？”

校服周尧立刻说：“我怕黑。”

白背心周尧沉默。

马小满小声说：“周尧不怕黑，他小时候还钻防空洞吓人。”

校服周尧的脸僵了一下。

沈星河问白背心周尧：“你为什么说别让他回来？”

白背心周尧抬头。

他的眼睛里有很深的疲惫。

“因为他已经学会我妈的声音了。”

林照野后背发冷。

校服周尧笑了。

“我只是想回家。”

阿铃突然开口：“两个都不能直接带回去。”

顾南舟猛地看她。

“什么意思？”

“活人进了灯市街，会被影子缠住。影子学了活人，也会长出想回家的心。”阿铃看着两个周尧，“一个是被困住的人，一个是被认出来的影。现在他们连在一起。你们认错，影会回家。你们全不认，活人也会迷路。”

马小满崩溃地看天：“那到底怎么办？”

阿铃说：“灭灯。灯灭了，学来的东西会掉下去，剩下的才是人。”

校服周尧脸色变了。

他往后退一步。

牌楼后的灯市街响起无数门开的声音。

归人们同时抬头。

阿铃开始唱。

“山下有城，城下有灯。”

“一更点灯照旧路。”

她的声音很稳。

林照野他们跟着往前冲。

引魂灯在牌楼后的祭台上。

祭台不高，像一口倒扣的井。中央放着一盏青铜灯，灯芯细得像一根黑发，火苗只有豆大，却把整座地下青梧照得通明。

火光里，林照野看见无数张脸。

周尧妈妈，马小满妈妈，顾南舟爷爷，沈岚，林成川，赵砚秋，所有想让某个人回来的人，都像被火苗照在里面。

原来灯不在井里。

灯在人心里。

林照野从书包里掏出黑布。

就在这时，火苗里传来林成川的声音。

“照野，别盖。”

林照野的手停住。

火苗里出现年轻的林成川。他站在暗房红灯下，手里拿着一张照片。照片上是小时候的林照野。

“灯灭了，我就再也回不去了。”年轻的林成川说。

“你已经回来了。”林照野咬牙。

“不完整。”火里的父亲看着他，“你不想知道我留在那边的那一部分是什么吗？”

林照野的手开始发抖。

他想。

他太想了。

这些年，他一直想知道父亲为什么变成这样，想知道那个会笑、会带他去拍照、会把他举到肩上的林成川到底去了哪里。

沈星河突然握住他的手腕。

“别一个人看。”

她的声音很轻，却把林照野拉了回来。

火苗里又响起沈怀川的声音。

“星河。”

沈星河没有看火。

她盯着林照野，眼泪流下来，手却没有松。

“盖。”

顾南舟冲过来，挡住从后面扑来的校服周尧。

校服周尧的脸已经开始变。五官一会儿是周尧，一会儿是顾北川，一会儿又变成马小满外婆。它急得像终于露出本来的空。

白背心周尧跪在地上，像被什么线拴着。

“快！”他喊。

阿铃的唱声忽然断了一下。

归人群里，一个穿白衬衫的男人走出来。

沈怀川。

沈星河终于看见了他。

男人站在灯光边缘，脸色苍白，却看着她笑了一下。

“长这么大了。”

沈星河整个人僵住。

马小满突然拿出复印页，哭着接上唱：

“三更莫问来时处！”

他的声音跑调，发抖，难听得要命。

但唱词接上了。

阿铃回头看了他一眼，继续唱：

“四更莫喊旧时名。”

沈星河闭上眼。

“爸，对不起。”

她转身，和林照野一起把黑布盖向引魂灯。

火苗尖叫起来。

那不是人的声音。

黑布落下的一瞬间，整座灯市街的灯同时灭了一半。

校服周尧摔在地上，身体像湿纸一样塌下去。白背心周尧也倒下，胸口剧烈起伏。

顾南舟扑过去，把白背心周尧拖出来。

“他有呼吸！”顾南舟喊。

林照野还没来得及松口气，祭台下面忽然裂开一道缝。

黑布下的引魂灯没有完全灭。

火苗从布缝里钻出一点。

阿铃脸色变了。

“不够。”

“什么不够？”马小满喊。

“灯芯还在。”

阿铃走上祭台，伸手去抓灯芯。

林照野抓住她：“会怎么样？”

阿铃看着他。

“守灯的人，要和灯一起走。”

“不行。”沈星河说。

阿铃笑了一下。

那是他们第一次看见她像个真正的小孩。

“我本来就不是来回家的。”

她伸手，拔下灯芯。

雾岭所有灯火同时向她身上涌去。

阿铃的青布褂被光照得几乎透明。她提着那只灯笼，灯笼终于彻底亮起。里面不是火，是一条很窄很窄的路。

“带活的走。”她说。

然后，她连同那盏灯笼一起，落进祭台裂缝。

地下青梧开始塌。

不是房屋倒塌，而是所有记忆一起褪色。门牌反转，招牌落下，纸扎铺的人影散成灰。归人们站在雾里，有的哭，有的笑，有的第一次露出不属于模仿的茫然。

林照野他们拖着周尧往回跑。

背后，沈怀川的声音最后一次响起：

“星河，别回头。”

沈星河没有回头。

她跑得很快。

山神庙的门在他们身后合上时，青梧县的电也回来了。

远处县城一盏盏亮起。

像终于有人把现实重新拧回原位。

周尧躺在庙前石阶上，咳出一口黑水。

周尧妈妈从山路上跌跌撞撞跑来，扑到他身边。

这一次，周尧没有叫妈。

他先茫然地看了她很久。

然后哑着嗓子说：

“我饿。”

马小满瘫坐在地上，哭着笑了。

“是他。”他说，“这缺德话是活人说的。”
