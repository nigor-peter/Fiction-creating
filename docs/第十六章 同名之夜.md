# 第十六章 同名之夜

沈星河不记得他们了。

她记得怎么拆收音机，记得青梧县医院每条走廊，记得第七号实验井的频率，甚至记得父亲最后那句“不要来找我”。

可她看着林照野，叫不出名字。

看着马小满，她只知道这个人话多。

看着顾南舟，她知道他有个哥哥，却不知道哥哥叫什么。

她也不记得自己是谁。

“沈星河。”林照野一遍遍告诉她，“你叫沈星河。”

她摇头。

“广播也这么说。”

“广播在抢你的名字。”

“名字怎么会被抢？”

马小满举起纸灯笼：“能抢。这里什么缺德事都干得出来。”

阿铃从灯笼上撕下一张空纸，让沈星河自己写名字。

她握住铅笔。

“沈”字写到一半，笔画忽然变成另一个名字：沈怀川。

沈星河盯着纸，眼泪毫无预兆地落下来。

她不记得伙伴，却仍记得父亲。

第二盏灯正在用最深的记忆替换现在的身份。

阿铃说：“要在天亮前回现实。名字被叫三次，就会在这里留下位置。”

“刚才叫了几次？”顾南舟问。

“一次。”

旧城上方传来广播。

“沈星河。”

第二次。

他们带着沈星河往修堤船方向跑。

无名者没有追。他们站在街边，羡慕地看着她。对没有名字的人来说，被叫三次不是灾难，是梦寐以求的事。

周尧忽然停住。

前方站着校服周尧。

这一次，他有影子。

影子从脚下拖出很长，连接到周尧脚边。

“名字给我。”校服周尧说，“你已经用过一次了。”

周尧的脸色惨白。

“你不是没了吗？”

“影子不会没。”对方说，“只要还有人记得你在灯市街的样子。”

校服周尧伸出手。

现实周尧胸前慢慢浮出自己的名字，像湿纸贴在皮肤上。校服周尧一拉，“周尧”两个字便从他胸口翘起一角。

顾南舟扑过去，一拳打在影子脸上。

拳头像打进冷水。校服周尧的脸散开，又迅速恢复。

“别打他！”沈星河突然喊。

她虽然不记得周尧，却看懂了线路。

“他们共用一个名字。你伤影子，真的也会受伤。”

现实周尧鼻子里流出血。

校服周尧笑了。

马小满从纸灯笼上撕下一张纸，飞快写字。

“周大锤。”

他把纸贴到校服周尧胸口。

影子愣住。

马小满又写一张：“周欠账。”

第三张：“周不还弹珠。”

他一张张往影子身上贴。

“你不是想要名字吗？我多给你几个！”

无名者群里传来轻轻的笑声。

校服周尧胸口被错误名字盖满，和现实周尧之间的影线开始混乱。顾南舟趁机把周尧拖开。

马小满转身就跑：“我只是暂时骗住它，回头它想明白肯定很生气！”

他们终于在旧戏台外找到修堤船。

老吴坐在船头，却不认识自己的船。

“你们是谁？”他问。

他的失名已经更深，连万灯巷和水库都忘了。

阿铃接过竹篙。

她把船撑上水面。

现实青梧也乱了。

水库管理所的档案、县医院病历、学校花名册和户籍登记同时出现改名。两个同名的人开始交换记忆。

粮站的两个陈国梁，一个记得另一个人的妻子，一个认得另一个人的家门。学校三个叫李军的学生互相写对方暑假作业。县医院里，一个刚出生的孩子病历上出现了七十岁老人的名字。

最严重的是照相馆。

林成川洗出的每一张证件照上，客人的脸都变成了另一个同名者。

林照野他们回到岸上时，沈岚已经等在那里。

她冲过来抱住沈星河。

沈星河却僵着身体。

“您是医生。”她说，“我见过您。”

沈岚的脸一下子碎了。

“星河，我是妈妈。”

广播第三次响起前，阿铃把纸灯笼罩在沈星河头顶。

马小满坐在地上，把所有记得的事写在灯笼纸上。

沈星河，十三岁。

会修收音机。

不爱吃肥肉。

写字很整齐。

说谎的时候会先抿嘴。

爸爸叫沈怀川，妈妈叫沈岚。

朋友是林照野、马小满、顾南舟、周尧。

“最后一句不用写我。”顾南舟说。

“都什么时候了你还嫌肉麻？”马小满骂他，“一个都不能少。”

林照野按住灯笼，沈岚握住女儿的手，所有人一起叫她。

“沈星河。”

县广播也第三次叫出：

“沈星河。”

两个声音撞在一起。

纸灯笼从中间裂开。

沈星河倒在母亲怀里。

过了很久，她睁开眼。

第一眼看见林照野。

“你的脸怎么了？”

林照野抬手摸脸。

什么都在。

沈星河却从地上捡起他的相机，翻到刚才误触拍下的一张照片。

照片里有两个林照野。

一个站在水库岸边，扶着纸灯笼。

另一个站在水面下的旧城牌楼前，胸口贴着“林照野”三个字。

两个都有脸。

两个都有影子。

照片背面，出现一句话：

同名者，只能留一个。
