# 第十八章 叫名的人

赵砚秋站在沉城戏台中央。

他换了一身旧长衫，头发梳得整齐，像文化馆照片里那些早已死去的说书先生。第二盏灯放在他面前，灯火不是黄色，而是近乎白色。

台下坐满无名者。

每个人膝上都放着一块空木牌。

赵砚秋翻开县志残卷。

“第一位归人，赵砚萤。”

灯火猛地升高。

现实青梧县里，一个叫赵砚萤的老太太正在家中吃饭。她忽然放下碗，不认识自己的儿女，也不记得自己姓什么。

名字离开现实，沿水库电线、病历、户籍纸和旧照片向沉城流去。

戏台帘幕后，传来小女孩的脚步声。

顾南舟按走边步法冲向台心。

赵砚秋抬手，数十块姓名牌飞起来，挡住他的路。每块牌上都浮出顾北川的名字。

“你不想让哥哥回来？”赵砚秋问。

“想。”顾南舟挥木刀劈开姓名牌，“但想不是拿别人的命换。”

“只是名字。”

“没名字的人，还算活着吗？”

赵砚秋的动作停了一瞬。

戏台地下，沈星河拆开收音机。

“点名通过旧频段传到现实。”她说，“我能切断。”

“怎么切？”马小满问。

“烧掉线圈。”

“烧了你以后就听不到你爸了。”

沈星河看着收音机。

这是父亲留给她的最后一条路。只要频段还在，她总觉得某一天还能听见沈怀川，哪怕只是旧录音，哪怕只是灯学出来的反话。

头顶传来赵砚秋第二次唱名：

“沈怀川。”

收音机自己亮起。

父亲的声音传出来：

“星河，别烧。”

沈星河闭上眼。

“你真的话，不会这么说。”

她把线圈扯出来，按到旧戏台灯火上。

铜线烧红。

收音机里所有声音同时尖叫。

沈怀川最后喊了一声她的小名。

沈星河手抖得厉害，却没有松开。

线圈熔断。

现实县城广播全部哑了。

赵砚秋手中的县志残卷燃起一角。他后退一步，第一次露出慌乱。

马小满抱起点名册：“现在怎么办？”

“把已经被叫的名字划掉。”沈星河说。

“划掉他们会怎样？”

“不知道。”

马小满握着铅笔，迟迟不敢落下。

他不是怕鬼。

他是怕自己一笔下去，真的把某个人从世界上删掉。

无名者围上来。

他们没有攻击，只伸出手，想摸那些名字。

马小满突然明白：名单不是用来删的。

他翻到空白页，开始抄。

赵砚萤。

沈怀川。

顾北川。

不是把名字留给第二盏灯，而是把它们写进活人的记录里。

“你们不用抢。”他一边写一边说，“我记。写不下我就回家找本子。一本不够买十本，我家有货。”

无名者停住。

空木牌上开始出现各自残缺的笔画。

台上，赵砚秋叫出第三个名字：

“顾北川。”

顾南舟脚下的方格全亮了。

顾北川站在台心另一侧，身体比之前清楚。他看向弟弟，摇了摇头。

“别过来。”

这一次，顾南舟相信这是哥哥。

因为真正的顾北川从来不会要求他冒险证明爱。

赵砚秋翻到第四页。

“林照野。”

另一个林照野在戏台地下睁开眼。

他从红光里扑出来，抓住点名册。两个林照野同时按住同一个名字。

“你没有我完整。”影子说。

“我本来就不完整。”林照野回答。

他想起父亲说过的话。

缺了一块，也能活。

林照野主动撕下写着自己名字的那一页。

影子大笑：“没有名字，我们都会消失。”

“那就让别人叫。”

林照野看向伙伴。

沈星河第一个说：“林照野。”

马小满跟着喊：“林照野。”

顾南舟在台上喊出他的名字。

周尧的声音从水面上方传来。

林成川、沈岚、马莲，甚至病床上的人和老街上的孩子，都在现实里叫他。

名字不是纸上的位置。

是人与人之间的回答。

另一个林照野胸口的字开始脱落。

他脸上的愤怒变成害怕。

“那我呢？”

林照野看着他。

这个影子有自己的恐惧。

它已经不再只是模仿。

“你不能用我的名字。”林照野说，“但你可以给自己取一个。”

影子愣住。

台上突然传来女孩的声音。

“哥哥。”

所有人抬头。

戏台帘幕被一只小手掀开。

赵砚萤走了出来。

她和照片里一模一样，梳两条短辫，穿白布衫，赤着脚。她知道赵砚秋所有秘密，知道他小时候怕雷，知道他把妹妹的糖葫芦偷吃后藏在井边。

赵砚秋哭着往前走。

“砚萤。”

女孩看着他。

她没有叫哥哥。

她身后也没有影子。
