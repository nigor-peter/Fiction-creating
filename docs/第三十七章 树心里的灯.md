# 第三十七章 树心里的灯

他们最终还是进了无声林。

沈岚没有拦。

她只把一只旧电筒递给沈星河，又在她掌心放了一枚已经不亮的电池。

“你爸当年也是这样走的。”沈岚说，“他说科学要有证据。我后来才明白，有些证据会要人付账。”

沈星河把电池收进兜里：“我去不是为了把他拖回来。”

沈岚眼眶红了，却点头：“那就去听完他没说完的话。”

无声林的入口在林场后山。

没有雾。

这比有雾更可怕。树一棵挨一棵站着，叶子不动，鸟不叫，连脚踩落叶都没有声音。林照野把相机挂在胸前，却迟迟没有举起来。

阿铃走在最前面。

她现在不确定自己是不是阿铃，也不确定岑小铃这个名字该不该拿回来。那个名字像一件旧衣服，针脚还在，却不知道穿上后是不是还能走进现在的教室。

何栖在一棵树下等他们。

“你们带灯了吗？”他问。

沈星河摇头。

马小满小声说：“带了也没用，这地方连我肚子叫都没声。”

何栖看了他一眼：“你还记得我吃过什么糖？”

“桂花糖。”马小满立刻说，“两块。你还欠五分钱。”

何栖愣住。

过了很久，他说：“我没有钱。”

“没事。”马小满摆摆手，“算你赊着。等出去再还。”

树影动了一下。

像有什么东西因为这句普通得不能再普通的话，松开了一点。

他们继续往里走。

林子深处有一棵巨大的空心树。树干粗得五个人合抱不过来，中间裂开一道缝，里面没有黑暗，只有一圈圈极细的光。

那些光像年轮。

一圈是引魂灯的黄。

一圈是司南灯的青。

一圈是星钉的白。

还有更多他们没见过的火色，层层嵌在树心里，像青梧所有灯路都曾从这里取走一点光。

沈星河的收音机忽然亮了。

没有声音的林子里，第一次响起电流声。

沈怀川从树后走出来。

他比沈星河记忆里的照片瘦很多，头发白了一半，衣服还是失踪那年穿的衬衫。可他的影子扎在树根里，像一截无法拔出的木桩。

沈星河没有喊。

她只是看着他。

沈怀川也看着她，眼神里的疼比任何拥抱都重。

“你长高了。”他说。

沈星河笑了一下，眼泪同时掉下来。

“你倒是没怎么变，除了像没好好吃饭。”

沈怀川低头看自己的手：“我不能离开这里。”

“我知道。”

这句话让沈怀川愣住。

沈星河把那枚没电的电池放到树根上：“妈让我听完你没说完的话。”

沈怀川沉默很久。

“当年实验打穿的不是地下水脉，是无声林借给青梧的灯芯路。我们以为找到了异常磁场，实际上是找到了一笔旧债。”

“你留下，是为了守住它？”

“我留下，是因为我第一个听见那些名字。”沈怀川说，“只要我的声音还在这里当灯标，青梧还能听见自己欠过什么。”

沈星河握紧手。

林照野看见她口袋里的星钉亮起来。

星钉没有指向出口。

它指向阿铃。

阿铃站在树心光里，空白姓名牌一点点浮出旧字。

岑小铃。

何栖看着她：“你是第零生产队最后的铃。”

阿铃抬头，声音很轻：

“那阿铃是谁？”

树心里的灯全部颤了一下。

沈怀川回答不了。

因为这个问题，只能由她自己回答。
