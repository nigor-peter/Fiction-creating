# 第十五章 阿铃的来处

阿铃是在万灯巷十七号出现的。

那是一间没有门的旧屋。墙上挂着许多灯笼骨架，竹篾已经发黑，红纸全被水泡烂。屋中央放着一只木盆，盆里没有水，只有一截烧焦的灯芯。

阿铃坐在木盆边，低头缝灯笼。

她用的不是针线。

是一根红绳，把一张张写过名字的纸缝在一起。

“你还活着。”马小满脱口而出。

阿铃抬头，认真想了一会儿。

“活着是什么意思？”

几个人的心都沉了下去。

她开始忘记现代词语。

沈星河蹲在她面前：“你记得我们吗？”

阿铃依次看过他们。

“照相的。”

“听匣子的。”

“怕鬼的。”

“唱武戏的。”

轮到周尧，她停了很久。

“回来过两次的人。”

马小满松了一点气：“至少‘怕鬼的’还算准确。”

林照野把赵砚萤的照片放到她面前。

阿铃看了一眼，摇头。

“她不在这里。”

“赵砚秋说欠她一个名字。”林照野说。

“名字不是欠的。”阿铃继续缝纸，“是别人叫你，你回头，才有的。”

沈星河问：“这些无名者为什么只记得你？”

阿铃的手停住。

屋里的灯笼骨架轻轻碰撞。

她看向门外那条沉在水底的旧街。

“从前，我叫铃娘。”

这是她第一次说出自己的旧名字。

清末那年，青梧县发大水。

山洪冲走半条街，死者和失踪者多得来不及入册。没有名字的人不能进祠堂，也没有家人为他们点灯。灯戏班便收集遗物，在七月半唱“送灯戏”，替无人认领者走完最后一段路。

铃娘是戏班最小的孩子。

她负责提最后一盏灯。

送灯那晚，回头井突然开了。无数无名者跟着灯上岸，没有家门可回，只能围着戏台叫“给我一个名字”。大人们怕了，丢下灯逃走。

只有铃娘没有跑。

“为什么？”顾南舟问。

阿铃说：“他们都没有人记得。我要是也走，就真的没有了。”

她把自己的名字写在灯笼里，留在井边。

从那以后，无名者记不得自己，却都记得提灯的小女孩。时间久了，“铃娘”变成“阿铃”，她也成了送灯人。

“所以你不是守着门不让他们出来。”沈星河说。

阿铃点头。

“我是送他们去该去的地方。”

“哪里才是该去的地方？”马小满问。

“有人记得，就回记忆里。没人记得，就走灯路。”

“灯路通到哪？”

阿铃看着他。

“我不知道。”

这答案反而让马小满安静了。

林照野问：“赵砚萤呢？”

“她没有走灯路，也没有进地下青梧。”阿铃说，“山洪那晚，她被下游人家救起，后来去了别处。”

所有人都愣住。

“她可能活过？”沈星河问。

“活过很久。”

“赵砚秋不知道？”

阿铃摇头。

林照野突然明白了第二卷最残忍的地方。

赵砚秋等了一辈子的妹妹，也许从来没有被困在另一个世界。真正困住他的，是没有答案的那场山洪。

“第二盏灯能叫回什么？”顾南舟问。

“只能叫回他记得的妹妹。”阿铃说，“不是后来活过的人。”

屋外传来无名者的脚步声。

他们聚在万灯巷里，胸前的木牌仍写着阿铃。第二盏灯虽然被赵砚秋带走，失名却没有停止。

阿铃把缝好的纸灯笼递给马小满。

“拿着。”

“为什么给我？”

“你会记。”

马小满看着灯笼。每一片纸上都只有半个名字，有的缺姓，有的缺名，有的只剩乳名。

“我记不住这么多人。”

“记不住也要写。”阿铃说，“写错可以改，不写就没有了。”

马小满抱住灯笼，第一次觉得一件东西比他的书包还重。

沈星河问：“你会跟我们走吗？”

阿铃看着她，像没听懂“走”这个现代而普通的词。

过了一会儿，她站起来。

“我要去阻止点名。”

他们回到旧戏台时，修堤船已经不见了。

老吴也不见了。

青石板上只留下竹篙拖过的痕迹，一直通向沉城更深处。

沈星河的收音机重新响起。

县广播女播音员正在播报失物招领：

“现有一批姓名无人认领，请下列居民前往旧青梧沉城戏台领取。”

她开始念名字。

每念一个，现实中的某处就传来玻璃碎裂声。

阿铃脸色变了。

“他开始叫名了。”

广播最后念出一个名字。

“沈星河。”

沈星河手里的收音机掉在地上。

她抬起头，眼神茫然。

“你们是谁？”
