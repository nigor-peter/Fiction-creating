# 第十一章 第二盏灯

周尧忘记马小满名字的那天，青梧县刚下完一场雨。

雨不大，从凌晨落到清早，把老街洗得发亮。屋檐还在滴水，电线上的麻雀挤成一排，小卖部门口那块“冰袋五角”的硬纸牌被泡软了一角。

马小满提着两根油条去县医院看周尧。

他推开病房门的时候，周尧正坐在床边穿鞋。床头柜上放着半碗稀饭、一只削到一半的苹果，还有那张已经被揉皱的寻人启事。

“你能下床了？”马小满把油条往桌上一放，“我还以为你得躺到开学，正准备继承你的暑假作业。”

周尧抬头看他。

眼神很陌生。

马小满脸上的笑慢慢停住。

“怎么了？”

周尧皱着眉，像在一道很难的题目里找答案。

“你是……”

“你别开玩笑。”

周尧没有笑。

他盯着马小满看了很久，脸上渐渐露出着急的神色。

“我认识你。”他说，“你家开小卖部，你怕黑，五年级体育课跑八百米的时候装肚子疼。可你叫什么？”

马小满站在病床前，一句话也说不出来。

窗外屋檐落下一滴雨。

啪。

像有人在很远的地方，轻轻点了一下名册。

半小时后，林照野、沈星河和顾南舟都到了。

沈岚给周尧做了检查。体温正常，瞳孔正常，记忆也没有整段缺失。他记得第一章放假那天班主任扔过来的粉笔头，记得顾南舟要酸梅味冰袋，记得林照野在水泵房拍了照片。

他只是不记得马小满的名字。

“可能是受惊后的暂时性记忆障碍。”沈岚说。

她说得像个医生，语气平稳。可她写病历时，笔尖停在姓名栏上很久。

姓名：周尧。

那两个字像在纸上轻轻晃动。

沈星河看见了。

“妈。”

沈岚立刻合上病历本。

“让他休息。”

她走出病房。

顾南舟等门关上，才低声问：“你还记得灯市街吗？”

周尧的脸色一下子变白。

“记得一点。”

“两个你呢？”

“别问了。”林照野说。

周尧却看着自己的手。

“那个穿校服的，还在。”

马小满往后退了一步：“在哪？”

周尧抬起右手，指了指自己的太阳穴。

“这里。”

病房里安静了。

“有时候我睡着，他会替我做梦。”周尧说，“梦里有很多门牌。每块门牌上不是号码，是名字。他一直找你们的名字。”

“为什么找我们的？”沈星河问。

“他说自己没有名字了。”

周尧抬头看向马小满。

“昨晚他找到一个。”

马小满的嘴唇动了动。

“我的？”

周尧点头。

门外传来护士推车经过的声音。轮子碾过地砖缝，一下一下，像旧戏台的鼓点。

林照野忽然想起第十章最后那张照片。

第一盏灯已灭。

第二盏灯将醒。

“去照相馆。”他说。

林成川正在给人拍证件照。

客人是粮站会计，一个四十多岁的男人，穿白衬衫，头发梳得很整齐。他坐在蓝色背景布前，林成川让他看镜头，他却不停回头看门口。

“师傅，”男人不安地问，“我刚才登记写的什么名字？”

林成川放下相机。

“陈国梁。”

“对，陈国梁。”男人松了口气，又重复两遍，“陈国梁，陈国梁。”

可当照片从暗房里洗出来时，照片下面的姓名纸牌是一片空白。

男人盯着自己的照片，脸上没有血色。

“我明明写了。”

林成川没有收钱，让他先回家。

男人走后，四个孩子从后门进来。马小满一路都在念自己的名字，念得顾南舟终于忍不住。

“你再念，我先忘了你。”

“这叫加深印象。”马小满说，“马小满，马小满，马小满。小马吃得满，名字忘不完。”

没人笑。

林照野把第十章最后那张照片拿出来。

照片一直锁在铁皮盒里。七天过去，牌楼、阿铃、沈怀川、顾北川和年轻赵砚秋都没有变化。只有背面那两行水字淡了。

林照野把照片放进清水。

水面先是平静。

几秒后，照片边缘冒出一串细小气泡。背面的旧字消失，新的水痕慢慢浮出来：

旧名归位，活名让路。

沈星河念了一遍。

“第二盏灯管名字。”

林成川站在暗房门口，脸色一点点沉下去。

“不是名字。”他说，“是位置。”

“什么位置？”顾南舟问。

林成川拿起那张空白姓名牌的证件照。

“一个人活在世上，要有别人叫他，有纸写他，有照片证明是他。第二盏灯要让死去的人重新占住这些位置，就得先把活人的名字腾出来。”

马小满捂住胸口：“它为什么先找我？”

周尧坐在一旁，低声说：“因为我记得你。”

所有人都看向他。

“校服那个我，是从我们的记忆里长出来的。”周尧说，“他认识的人越多，就能摸到越多名字。”

林照野看向父亲：“赵砚秋呢？”

林成川摇头。

赵砚秋从七月十五之后就没出现过。

文化馆说他请了病假，家里没人，办公室锁着。那张照片里，年轻赵砚秋手持点燃的灯，像是在告诉他们，他的失踪不是意外。

四个人赶到文化馆时，门口看门大爷正拿着登记本发愁。

“怪了。”他说，“赵老师叫什么来着？”

顾南舟一把抓住登记本。

过去十年的来访记录都在。每一页都有赵砚秋的签字。可现在，所有签字只剩一个“赵”字，后面两个字像被水泡走了。

他们冲上三楼。

赵砚秋办公室的门开着。

书架少了一整排资料，桌上的樟脑盒被打翻，地面留着一串湿脚印。脚印从书柜走到窗边，却没有离开的方向。

沈星河检查桌面。

“县志残卷不见了。”

顾南舟拉开抽屉，里面只有一张借阅卡。

资料名：青梧旧城搬迁图。

去向：水库管理所。

借阅人那一栏没有名字。

只有一枚小小的灯形印章。

林照野举起相机，拍下办公室。

快门落下的一瞬间，他听见身后有人叫他。

“照野。”

声音是林成川。

他差点回头。

可他爸根本没有跟来。

林照野僵在原地，慢慢把相机从眼前放下。

办公室玻璃窗上映出一个孩子。

那孩子站在他身后，和他一样高，一样瘦，脖子上也挂着海鸥相机。

唯一不同的是，他脸上没有五官。

只有额头上写着三个湿漉漉的字。

林照野。

玻璃里的孩子抬起手，把那三个字从自己脸上揭下来。

然后贴在胸口。

林照野猛地转身。

身后什么也没有。

回到照相馆后，他把刚拍的胶卷洗出来。

照片里，赵砚秋办公室空空荡荡。沈星河弯腰看桌面，顾南舟站在书柜边，马小满正翻登记本。

窗户旁还站着一个孩子。

没有脸。

林照野把照片凑近红灯。

无脸孩子胸口贴着他的名字。

而照片右下角，真正的林照野站在阴影里。

他的脸还在。

名字却没有了。
