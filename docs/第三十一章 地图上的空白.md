# 第三十一章 地图上的空白

地理课是在第二节。

青梧小学的教室还带着早晨没散尽的潮气，窗台上摆着几盆快死的太阳花。老师把新挂图展开时，竹竿敲在黑板上，发出一声很脆的响。

“今天讲我国地形。”

全班本来都在翻书，只有林照野抬头看了一眼。

那张地图不是昨天那张。

西北角多出一块灰绿色的林区。没有省界，没有河流，没有公路，边缘像被水泡过，晕开一圈浅白。图例上写着四个字：未勘测林区。

阿铃坐在最后一排，铅笔停在半空。

周尧轻轻碰了碰林照野的胳膊：“又大了。”

林照野记得很清楚，昨天那块灰绿色只有指甲盖大。今天它已经像一块湿布，盖住了地图西北角的山脉。

老师没有发现异常。他把粉笔头在地图上一点，讲盆地、丘陵、河流走向。讲到青梧所在的西南山区时，学校广播忽然“滋”了一声。

全校喇叭一起响。

不是音乐，也不是通知。

是一段空白。

那空白不是安静。它像一只看不见的手按住了每个人的耳朵，连窗外的蝉声、自行车铃、操场上低年级的吵闹，都被一起按没了。

老师还在张嘴。

没有声音。

全班慢慢安静下来。马小满把铅笔摔到桌上，铅笔弹了一下，他脸色立刻白了。

铅笔也没有声音。

沈星河从书包里摸出小收音机，拧开开关。指示灯亮着，频率指针轻轻颤，里面却只有一团棉絮似的静。

只有阿铃站了起来。

她看向窗外。

林照野顺着她的目光看过去，学校后面的香樟树在秋风里晃。树干上像有人用指节敲了三下。

咚。

咚。

咚。

声音只落在他们几个人耳朵里。

广播突然恢复。

老师的后半句话撞出来：“……所以山地交通不便。”

全班像被水冲醒，椅子腿一阵乱响。没人记得刚才失声的几秒，只有马小满抓着桌角，嘴唇还发白。

下课后，林照野把地图拍了下来。

他没有用闪光灯。旧相机“咔嚓”一声，快门声这次正常得让人心里发紧。

中午，他跑回照相馆冲照片。

林成川正在柜台后擦镜头，看见儿子从暗房里出来，手上还滴着药水，脸色就沉了下来。

“又拍到东西了？”

林照野把照片放在红灯下。

照片里的地图比教室里那张更清楚。灰绿色林区中间，多出一条白线。白线像一条新修的公路，从青梧县城西北绕进山里，尽头有一个小小的黑点。

林照野眯起眼。

黑点旁边有字。

青梧县第零生产队。

“我们县有这个地方吗？”周尧问。

没人回答。

阿铃伸手想摸照片，被林成川一下按住。

“别碰。”他说。

他的声音很低，像怕惊动照片里的树。

“爸，你知道第零生产队？”

林成川擦镜头的布停住了。他看着那张照片，看了很久，才说：“生产队从第一队排，哪有第零队。”

“可你听过。”

林成川没有否认。

傍晚放学时，周尧第一个发现花名册不对。

班主任把花名册忘在讲台上，封皮卷起一角。周尧本来只是想看自己有没有被记旷课，翻到他们班那页时，忽然不动了。

林照野凑过去。

花名册最后一行多出一个空白。

没有姓名，没有性别，没有出生年月，只有学号的位置写着一个“0”。

马小满干笑：“这谁啊？我们班还能招鬼插班？”

没人笑。

周尧指着那一行空白，手指微微发抖。

“昨天这里坐着一个人。”

“谁？”沈星河问。

周尧张开嘴。

他的表情先是困惑，接着变成恐惧。

他明明记得那个人趴在窗边写作业，铅笔盒是蓝铁皮的，桌洞里有一本没包皮的语文书。

可名字像被人从舌头上揭走了。

周尧说不出来。

教室后排，阿铃胸前的姓名牌轻轻晃了一下。

别针孔旁边，白塑料上裂开一条细细的缝。

窗外又传来三下敲树声。

咚。

咚。

咚。

这一次，整栋教学楼的灯同时暗了一下。
