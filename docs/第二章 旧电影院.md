# 第二章 旧电影院

周尧失踪后的第二天，青梧县所有人都在喊他的名字。

派出所的人喊，学校老师喊，周尧他妈喊，连街口卖凉粉的婆婆都跟着喊。声音从清早一直飘到中午，沿着老街、粮站、水渠和雾岭脚下的碎石路，一遍一遍地散开。

可没人再敢往雾岭深处喊。

大人们说那是因为山里容易有回声，喊乱了，反倒不好找人。

林照野知道他们在撒谎。

因为昨天晚上，雾岭已经回答过一次了。

早上七点不到，照相馆的卷帘门还没拉开，派出所的人就来了。来的是两个民警，一个姓刘，另一个年轻些，手里拿着本子。林照野坐在前屋的小圆凳上，鞋还没干，裤脚上结着泥，手指反复抠着相机皮套边缘。

“你们几个几点去的水泵房？”刘警官问。

“六点半。”

“谁提议去的？”

“周尧。”

“去干什么？”

林照野抬头看了他一眼。

这个问题昨晚他已经回答过。回答给老师，回答给周尧妈妈，回答给后来赶来的大人。每次他一说“灯”，对方的脸就会变一下，然后很快把话题带回去。

“他说雾岭有灯。”林照野说。

年轻民警笔尖停了一下。

刘警官看着他：“什么灯？”

“一排灯。黄色的，在雾里，会动。”

屋子里安静了两秒。

照相馆外面有人推着自行车过去，车铃响了一下，清脆得不合时宜。

刘警官把手里的搪瓷杯放到柜台上：“孩子，昨晚七点停电，天又暗。你们受了惊，可能把萤火、手电，或者山里找人的光看错了。”

“那时候还没人找他。”林照野说。

年轻民警抬头看他。

林照野继续说：“周尧就是追着那些灯不见的。”

刘警官没有立刻接话。他转头看向柜台后面。林成川站在那里，手里拿着抹布，一直擦同一块玻璃。玻璃已经很干净了，他还是在擦。

“成川，”刘警官说，“你儿子昨晚拍照了吗？”

林照野的心跳漏了一下。

林成川手里的抹布停住。

“没有。”他说。

林照野猛地看向他。

刘警官也看着林成川：“没有？”

“光线太暗。”林成川说，“孩子吓坏了，没拍成。”

林照野张了张嘴。

林成川没有看他。

刘警官又问了几句，都是走没走散、周尧最后在哪个方向、有没有听见水声。林照野一一答了。等两个人走后，卷帘门外的脚步声远了，照相馆重新静下来。

林照野站起来：“为什么撒谎？”

林成川把柜台上的搪瓷杯拿起来，发现不是自己的，又放回去。

“照片呢？”林照野问。

“收起来了。”

“我要给他们看。”

“不能给。”

林照野一股火冲上来：“周尧不见了！”

林成川终于抬头。他昨晚一夜没睡，眼睛里全是红血丝，脸色灰得像没冲洗干净的底片。

“正因为他不见了，照片才不能随便给人。”

“什么叫随便给人？他们是派出所！”

“他们不知道怎么看。”

林照野愣了一下，随即更生气：“照片还要怎么看？上面就是周尧！他说不定还活着！”

“你怎么知道上面那个是周尧？”

这句话像一盆冷水，兜头浇下来。

林照野站在原地，耳朵里嗡嗡响。

林成川声音低下去：“照野，有些东西被拍下来，不代表它就是真的。也不代表它能救人。”

“那你说怎么办？”林照野问，“什么都不做？”

林成川没有回答。

暗房门半开着，里面的红灯还没关。林照野忽然觉得那盏红灯像昨晚雾里的灯一样，暗暗地浮在屋子深处。

他转身冲出照相馆。

外面太阳很大。

老街上却没有往常放暑假的热闹。小卖部门口围了不少人，压低声音说话。一见林照野过来，又都不说了。马小满坐在门槛上，手里拿着一根没拆封的冰袋，已经化成了水。

“他们也问你了？”林照野问。

马小满点点头，脸色很差：“问了三遍。我说有灯，他们说我太紧张。我说有雾，他们说山里有雾正常。我说收音机里有周尧的声音，他们说……”

“说什么？”

“说我少看录像厅里的鬼片。”

林照野在他旁边坐下。

马小满用牙咬开冰袋角，却没吸。他盯着地面看了一会儿，小声说：“我昨晚回家以后，梦见周尧了。”

林照野转头看他。

“他站在我家小卖部门口，浑身都是水，问我买不买票。”马小满咽了一下，“我问什么票，他就把嘴张开，里面全是黑的。”

林照野没说话。

“我是不是吓傻了？”马小满问。

“可能吧。”林照野说。

马小满瞪他。

“但不止你一个。”林照野补了一句。

街对面，沈星河从县医院方向走过来。她背着书包，头发扎得很紧，脸上没有表情，但步子比平时快。

“顾南舟呢？”她问。

“不知道。”马小满说，“他爷爷早上拎着竹竿追了他半条街，估计在挨打。”

话刚说完，顾南舟就从老街转角出现了。

他没挨打，至少看起来没有。只是手背上缠着纱布，校服袖子卷到手肘，脸色比昨天更臭。他一走近，就把一个东西扔到小卖部门槛上。

红色拨浪鼓。

鼓面已经干了一些，红漆却比昨晚更暗，像被什么东西泡透了。两颗小珠子垂在两边，一动不动。

马小满立刻往后缩：“你怎么还拿着它？”

“不拿着，等大人扔了？”顾南舟说。

“扔了不好吗？”

沈星河蹲下来，没碰拨浪鼓，只看鼓柄底部。

“这里有字。”

林照野也蹲下。鼓柄下面确实刻着两个很浅的字，被泥塞住了一半。他用指甲把泥刮掉，看见第一个字像“青”，第二个字只剩半边。

“青什么？”顾南舟问。

马小满忽然说：“青影。”

三个人都看向他。

马小满脸色更白了：“我乱说的。”

“你知道？”林照野问。

“我小时候听我爸说过。”马小满不情不愿地说，“以前老街尽头有个青梧影剧院，大家都叫青影。放电影，也演戏。门口有人卖这种拨浪鼓，小孩进场前都要买一个，开场前摇，图个热闹。”

顾南舟皱眉：“我怎么没见过？”

“你当然没见过，早关了。”马小满说，“我爸说那里后来不干净。”

“怎么不干净？”

“我爸没细说。”马小满声音小下去，“只说有一年停电，电影放到一半，银幕上多出来一排人影。灯亮以后，前排少了三个小孩。”

风从小卖部门口吹过，塑料门帘轻轻碰在一起。

顾南舟说：“你现在编故事越来越像周尧了。”

马小满急了：“我没编！你问我爸！”

“问你爸，他会让我们别管。”沈星河说。

她从书包里拿出收音机，又拿出一盘小磁带。

林照野问：“你录了？”

沈星河点头：“昨晚回家前按下去的。声音很乱，但有一段能听。”

马小满立刻左右看：“别在这儿放，我妈耳朵比狗还灵。”

“去你家后面。”顾南舟说。

小卖部后面有个堆纸箱的小院，平时放汽水瓶、煤球和空竹筐。四个人挤在一堆纸箱旁边，沈星河把磁带塞进收音机，按下播放键。

先是长长的沙沙声。

像有人在黑暗里拖着一袋湿沙走路。

然后是锣鼓。

很轻，隔得很远。咚，咚，锵。咚，咚，锵。

马小满把冰袋捏得咔咔响。

杂音里传来沈星河昨晚的声音：“你们听见了吗？”

接着是顾南舟：“什么？”

再然后，那个声音出现了。

“我在这儿。”

是周尧。

马小满闭上眼：“别放了。”

沈星河没有停。她把音量拧大。

杂音起伏了一阵，里面忽然夹进一个陌生女人的声音。那声音像戏院里的报幕员，带着一种老式喇叭才有的空腔。

“青梧影剧院，晚场开映。”

林照野后背一凉。

报幕声之后，是很轻的一声拨浪鼓。

咚。

沈星河按下停止键。

小院里只剩远处街上的人声，还有汽水瓶被风碰出的细响。

“青影。”顾南舟看向红色拨浪鼓，“它在叫我们去旧电影院。”

马小满差点跳起来：“它叫你你就去？它要是叫你跳井呢？”

顾南舟看着他：“周尧可能在那里。”

这句话一出来，马小满没声了。

林照野也没声。

他想起照片里的周尧。湿透的校服，低着的头，手里那盏灯。还有最后那一下，像是抬起来看他们的脸。

“去看看。”林照野说。

马小满痛苦地仰头：“我就知道。”

沈星河把磁带收好：“白天去。带手电，不进地下，不乱碰东西。”

顾南舟已经往外走：“废话真多。”

沈星河说：“你手受伤了。”

顾南舟停了一下，没回头：“所以今天用脚踹门。”

青梧影剧院在老街最尽头。

那一带早就不热闹了。原先影剧院旁边有理发店、茶馆和一家卖布鞋的铺子，后来水库搬迁消息传了几年，年轻人往外走，老铺子一家家关门。现在只剩几个斑驳的门脸，铁锁生锈，门缝里长出细草。

影剧院的大门比林照野想象中更高。

门头上的字掉了两个，只剩“青  影  院”，中间空着，像被人抠走了。售票窗口被木板封住，海报栏里的玻璃碎了一半，里面贴着一张很旧的电影海报，人物的脸被太阳晒得发白，只剩一双眼睛还黑。

马小满站在台阶下，坚决不往上走。

“我在外面放风。”

顾南舟说：“你怕就直说。”

“我怕。”马小满说，“但我可以怕得很有用。”

沈星河走到售票窗口前，伸手摸了摸木板边缘。

“有人来过。”

林照野凑过去。木板下沿有新鲜刮痕，灰尘被蹭掉一片。像是有什么人，或者什么东西，从里面推过。

顾南舟从旁边捡起半块砖，砸开侧门上的烂锁。

“你不是说白天来就安全吗？”马小满问沈星河。

沈星河说：“我没说安全，只说白天。”

门被推开时，里面涌出一股冷味。

不是普通房子久不开门的霉味，而是一种潮湿、发酸、混着旧木头和老胶片的味道。大厅里很暗，阳光从破窗斜斜漏进来，照见空中飘着的灰尘。

林照野举起相机，拍了一张。

咔嚓。

马小满抖了一下：“你能不能别突然拍？听着像什么东西咬牙。”

“你可以在外面放风。”顾南舟说。

马小满看了看外面空荡荡的老街，又看了看里面三个人，最后骂了一句很小声的话，跟了进来。

大厅左边是售票窗口，右边是小卖部柜台。柜台里还摆着几个空玻璃瓶，瓶身蒙灰，商标早烂了。墙上挂着一块黑板，上面用粉笔写着模糊的字：

晚场七点。

林照野看着那四个字，心里一紧。

昨晚也是七点停电。

沈星河走到黑板前，伸手摸了一下粉笔字。

“不是新的。”她说。

“你怎么知道？”

“粉笔灰结块了。”沈星河收回手，“但有人碰过。”

黑板下沿有一道指印。

很细，很小，像小孩子的手指。

马小满不想看：“我们到底找什么？”

“周尧。”顾南舟说。

“我是说周尧要是真在这儿，会不会喊一声？”

就在他说完这句话的时候，售票窗口里传来一个声音。

“几张票？”

马小满整个人跳起来，撞倒一只空汽水瓶。瓶子咕噜噜滚出去，在地上打着转。

顾南舟一步冲到售票窗口前，抬脚踹开半块木板。

里面什么也没有。

狭小的售票间里堆着烂票根、蜘蛛网和一把断腿椅子。墙角有个铁皮喇叭，电线断了，像一条死蛇垂在墙上。

沈星河拿手电照过去。

“是喇叭。”

“断了还能说话？”马小满声音都劈了。

沈星河没回答。她蹲下来，在一堆票根里翻了翻，捡起一张。

票纸发黄，上面印着“青梧影剧院”。日期那一栏被水泡糊了，只剩时间看得清。

七点。

林照野发现票根背面有字。

不是印上去的，是用铅笔写的。字迹很浅，歪歪扭扭：

别回头。

顾南舟也看见了。他伸手要拿，沈星河却先把票根夹进书里。

“证据。”她说。

“你还真像派出所。”

“那你别乱踹。”

顾南舟没理她，转身往观众厅走。

影剧院的观众厅比外面看起来大。座椅一排排陷在黑暗里，红色椅套发霉开裂，像一张张烂掉的嘴。头顶的吊灯垂着，玻璃罩碎了两个。银幕挂在最前面，灰白色，中央裂开一道长口子。

四个人站在后排，一时间谁也没动。

马小满小声说：“这里以前真有人看电影吗？”

“废话。”顾南舟说。

“我是说，这么多椅子现在空着，比坐满还吓人。”

林照野举起相机，对准银幕。

取景框里，银幕中央那道裂口像一只闭着的眼睛。

他刚要按快门，沈星河忽然说：“等一下。”

她举起手电，光柱扫过座椅。

第三排中间，有一个座位上的灰被蹭掉了。

像有人刚坐过。

他们走过去。

座位扶手上放着一颗小珠子。

红色的，木头的，穿孔处还带着湿痕。

马小满看向顾南舟手里的拨浪鼓。

拨浪鼓左边的小珠子还在，右边的小珠子不见了。

顾南舟的脸色变了。

“刚才还在。”他说。

没有人说话。

林照野把那颗珠子拿起来。珠子很冷，冷得不像在夏天的屋子里放过。

忽然，观众厅前方传来咔的一声。

银幕亮了一下。

不是整张银幕亮，只是中央裂口附近泛起一块模糊的白光，像有人在后面点了一支很弱的灯。

马小满转身就想跑，被顾南舟一把拽住。

“不是我！”沈星河立刻说。

白光摇晃着，银幕上出现了影子。

先是老街。

灰蒙蒙的老街，路边挂着布招牌，街上没人。画面抖得厉害，像一卷被水泡过的旧电影。林照野认出来，那是青梧县的老街，可又不完全一样。街道比现在窄，房檐更低，影剧院门口挂着一排纸灯。

画面里走出一个孩子。

白背心，短头发。

周尧。

“周尧！”顾南舟往前冲。

银幕上的周尧没有回头。他沿着老街往前走，手里拿着一张票。街边那些关着门的铺子后面，好像站着很多人影，可一眨眼又没了。

林照野的心怦怦直跳。他举起相机，按下快门。

咔嚓。

闪光灯亮了一下。

银幕瞬间暗了。

顾南舟已经冲到台前，一拳砸在银幕旁边的木板上：“周尧！”

沈星河跑到放映室方向：“上面可能有机器。”

“我跟你去。”林照野说。

通往放映室的楼梯在观众厅后侧，很窄，扶手上全是灰。沈星河走在前面，林照野跟在她后面，手电光照着一级一级破损的台阶。马小满本来想留在下面，可一看顾南舟还在台前发疯似的扒银幕，立刻选择跟上。

放映室门半开着。

里面放着两台老式放映机，铁壳生锈，镜头蒙灰。墙边堆着几个胶片盒，其中一个盒盖打开，里面的胶片散出来，黑亮黑亮地垂在地上。

沈星河检查电线：“没通电。”

“那刚才银幕怎么亮的？”马小满问。

“我不知道。”

林照野走到放映机旁，发现镜头前卡着一小截胶片。他用手电从后面照过去，胶片上有模糊画面。

不是电影演员。

是周尧。

他站在一个井口旁边，低着头，像在往井里看。

井边立着一块木牌，牌上的字很糊，只能看清最后一个字：

井。

“下面！”顾南舟的声音从观众厅传来，“你们下来！”

三个人立刻往下跑。

顾南舟站在舞台左侧，手里抓着一截烂木板。银幕后面的墙被他掀开一块，露出一扇窄门。门上原本钉着木条，有几根已经断了。

断口很新。

门缝里吹出冷风，带着泥土味。

“这里有路。”顾南舟说。

马小满声音发颤：“你怎么知道这不是厕所？”

顾南舟把手电往里一照。

光没有照到墙。

门后是一段往下的台阶。

沈星河蹲下看地面：“有脚印。”

灰尘里确实有脚印。很小，杂乱，有些像孩子的，有些却拖得很长，像脚后跟没有抬起来。

林照野举起相机，想拍下来。

镜头对准门缝的一瞬间，他看见台阶深处有一点红。

像有人站在下面，手里拿着红色的东西。

他眨了一下眼。

红色不见了。

顾南舟已经钻进去。

“你等等！”沈星河压低声音。

“等什么？周尧可能下去了。”

林照野跟上去。马小满在门口站了三秒，最后咬牙骂道：“我上辈子肯定欠你们钱。”

台阶不长，却很陡。

下到底之后，是一条旧防空洞。洞顶很低，墙面潮湿，水珠顺着灰白色的水泥往下滑。手电光照出去，只能照见一截又一截弯曲的通道。空气里有土腥味，还有一股淡淡的煤油味。

墙上写着很多旧标语，大部分已经剥落。

备战。

防空。

深挖。

字下面还有后来乱画的粉笔印，歪歪扭扭，像小孩随手画的房子、小人和灯。

沈星河停在一面墙前。

“这不是乱画。”

林照野把手电照过去。

那是一张地图。

青梧县的老街、县医院、学校、粮站、水泵房，全都歪歪扭扭地画在上面。红漆干得发黑，像一条条凝住的血线。画地图的人不太会画，街道比例全乱了，可几个地方的位置又准得吓人。

林照野举起相机，忽然觉得后背发冷。

因为地图上，雾岭脚下多出了一条街。

现实里从来没有那条街。

街名旁边写着三个字：

灯市街。

而灯市街的尽头，还有一个小小的圆圈。

圆圈旁边，是另一行更小的字。

回头井。

马小满咽了口唾沫：“这井名听着……就不像让人活着回来的。”

没人笑。

因为就在他说完这句话的时候，防空洞深处传来了一声拨浪鼓响。

咚。

顾南舟举起手电。

光照不到尽头，只照见墙上那些红色街道，一条一条往黑暗里爬。

然后，又是一声。

咚。

这一次，声音后面还跟着一个很轻的声音。

“几张票？”
