# 第三十六章 无名戏

何栖的问题把所有人问住了。

知道一个名字，不等于认识一个人。

马小满抱着赊账本翻了一夜，翻到手指全是纸灰，才在夹层里找到一张旧糖纸。糖纸已经褪色，只有边角还能看出原来的红。

上面写着：桂花糖，两块。

没有落款。

马小满盯着那几个字，忽然想起他外婆以前说过，山里孩子最爱买桂花糖，因为能含很久，走山路时嘴里有点甜，就不怕雾。

“何栖吃过桂花糖。”他对空荡荡的户籍室说。

院子里没有人回答。

但那盏熄灭的路灯轻轻亮了一下。

第二天，顾南舟把他们带到县剧团旧戏台。

剧团早散了，门口贴着租赁广告，红纸被雨泡成暗褐色。后台还堆着旧盔头、刀枪把子和发霉的戏衣。顾南舟在箱底翻出一本线装戏谱。

戏谱没有名字。

封皮上只画了一只没有铃舌的铜铃。

“我哥以前排过这个。”顾南舟说，“我爷爷不让，说这不是给活人看的戏。”

林照野翻开戏谱，里面没有唱词，只有身段标记和锣鼓点：三慢、两急、一停。每一页旁边都写着动作：回头、举灯、听树、过桥、点名。

“无名戏。”赵砚秋站在门口，声音疲惫，“清末地方志里提过一句。不是招魂，是替不能说话的人留一个节拍。”

顾南舟没等他说完，就把戏谱往地上一放。

“那就演。”

“你一个人演不了。”沈星河说。

“谁说一个人？”

周尧第一个走上台。

他站在空荡荡的戏台中央，手心全是汗。

被救回来以后，他最怕别人看他。怕别人问他在地下青梧里到底是不是真的，怕有人在他身上寻找另一个周尧留下的缝。

可现在他忽然明白，比被怀疑更可怕的，是根本没人问。

“我来点名。”他说。

顾南舟敲第一下锣。

锣没有响。

台下灰尘却猛地震起一圈。

第二下，后台的旧幕布自己垂落下来。布面上没有影人，只有一排排空白脸。

第三下，阿铃胸前空白姓名牌亮了一瞬。

周尧开始念：

“何栖。吃过桂花糖。”

幕布上一张空白脸微微转向他。

“陈月苗。赊过红糖半斤。”

这个名字是马小满从账本压痕里拓出来的。

“罗大顺。借过草鞋一双。”

“岑小铃……”

阿铃猛地抬头。

这个名字不是他们找到的。

是周尧自己说出来的。

他也不知道为什么能说出口。像那个名字一直藏在他被地下青梧带走的空隙里，等到现在才有机会落地。

阿铃退了一步。

顾南舟的鼓点乱了。

幕布后面走过一道影子。

那影子很高，肩膀清瘦，手里拿着一盏没有火的灯。沈星河一眼就认出来，整个人僵在台下。

“爸。”

影子没有停。

他走过幕布，像走过一条很深的河。

沈星河追上台，用力掀开幕布。

后台空空荡荡，只有一只旧收音机摆在地上。

收音机没有电池，却亮着红点。

里面传来沈怀川的声音。

“星河，别进林子。”

短暂的电流声后，他又说：

“如果一定要来，不要带灯。”

阿铃站在戏台中央，脸上没有血色。

她摸着空白姓名牌，低声重复那个刚刚被周尧念出的名字：

“岑小铃。”

县剧团外面，天忽然阴了。

整条街的招牌灯一盏接一盏熄灭，像有人从远处吹过来一口气。

戏台上，没有响声的锣还在微微发抖。
