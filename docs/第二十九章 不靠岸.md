# 第二十九章 不靠岸

无岸处没有水。

无向船漂在一片纯黑里，船外没有上下，也没有远近。七座渡口缩成七个发光圆点，像七枚等待落下的棋子。

司南灯给出两个方案。

第一条：保留现在。

青梧、白鹭镇和其他渡口维持原状。失踪者的选择被保留，灾难也无法撤销。

第二条：重写灾难。

白鹭镇二十七人存活，陆家渡不被淹，沈怀川、顾北川等人可能走上其他路线。但周尧、马小满家、沈星河与伙伴们相识的经历，以及许多灾难之后才出生的人，将不再存在。

两条都是真的。

两条都有人受益，也都有人消失。

江晚站在第二条路线前。

她看见母亲不再一夜白头，看见二十七双鞋不必摆在镇政府门口，看见弟弟在河边长大。

江潮却站在第一条路线里，举着航标。

“姐，”他说，“你要救的是那天以前的我，还是现在的我？”

江晚说不出话。

“我现在会撑船，会认星，会给人指路。”江潮说，“你把山洪改掉，这些都没有了。”

“可你可以长大。”

“这也是长大。”

江晚终于明白，她一直把弟弟停在九岁那天。她爱的是需要自己保护的江潮，不是眼前这个已经做出选择的人。

她撕掉船票。

第二条路线没有消失。

愿望已经进入司南灯，撕纸不够。

沈星河把七枚星钉放在甲板上。

“让每个人只决定自己的方向。”

陆沉舟摇头：“船需要一个航向。”

“船需要，但人不需要。”

她提出共同校准：七座渡口各保留自己的现实，不再允许司南灯把某一条升级成唯一答案。代价是北斗渡会失去统一航道，以后每一艘船都必须得到目的地与乘客双方回应才能靠岸。

“会有更多人迷路。”陆沉舟说。

“也会少一些人被别人替他决定。”

阿铃把姓名牌压在第一枚星钉上。

江潮放上航标。

周尧放上寻人启事。

顾南舟放上半张戏票。

马小满放上点名册的一页。

林照野放下空胶卷盒。

沈星河最后放上烧坏的收音机线圈。

七件东西连成新的星图。

司南灯开始崩裂。

无向船也跟着裂开。木板下没有河，只有无数未选择的人生。掉下去的人不会死，却会永远在自己的可能性之间漂流。

陆沉舟走向舵位。

“我留下掌舵。”

“又来？”马小满抓住他衣服，“你们这些大人是不是特别喜欢用牺牲自己结束讨论？”

“总要有人。”

“那就轮流。”

“北斗渡上没有时间，轮流没有意义。”

林照野看向七枚星钉。

“不需要一个人握舵。让七座渡口各拉一条线。”

他们把七枚星钉钉进船舷。

每一枚都连接一个渡口。

船不再朝单一方向行驶。

它停在七条路线的张力中央。

不靠任何一岸。

也不抹掉任何一岸。

司南灯的火苗从一束分成七束。

船体裂缝停止扩大。

可最后一块甲板仍在下沉。

那里站着陆沉舟。

他的身体越来越透明。

“我没有渡口。”他说。

林照野拿出陆家渡照片。

“现在有了。”

照片上，村庄没有复活，但村口碑和一条通往北斗渡的小码头清晰可见。

陆沉舟第一次在水面留下影子。

远处传来鸡鸣。

无岸处开始天亮。
