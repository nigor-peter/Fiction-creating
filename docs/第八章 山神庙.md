# 第八章 山神庙

下午五点，青梧县的天已经黑得像傍晚。

云压在雾岭上，山脊只剩一条湿漉漉的影子。县城里家家户户都在找蜡烛、手电、煤油灯。小卖部门口排起长队，买火柴的人比买盐的人还多。没人承认自己害怕，可每个人说话都压着嗓子。

林照野他们在照相馆后屋集合。

桌上摊着阿铃给的山神庙图、实验井地图、归人名录、灯戏唱本抄下来的几句唱词，还有沈星河拆开的收音机。

林成川站在门口，像站在自己十年前没有跨过去的地方。

“山神庙后面不是引魂灯。”他说。

林照野看向他。

林成川慢慢走进来，从柜台底下取出一个铁皮盒。盒子打开，里面是一卷旧底片、一枚生锈钥匙，还有一块黑布。

“山神庙后面是灯路。”他说，“真正的引魂灯在雾岭旧祭台。想去祭台，必须从山神庙进。十年前，我们也是这么进去的。”

沈岚也来了。

她原本不肯让沈星河出门，可当县广播用周尧的声音说出“归人回家”后，她终于沉默地拿出医疗包和一只旧手电。

“我带你们到山神庙。”她说，“再往里，你们不许去。”

沈星河看着她：“你知道这句话没用。”

沈岚脸色苍白：“我知道。”

赵砚秋最后一个到。

他带来灯戏唱本的复印页和一串旧铜铃。

“唱本里说，进灯路时铃不能停。”他说，“铃停，路就会换。”

马小满盯着铜铃：“谁负责摇？”

所有人都看向他。

马小满后退一步：“我只是随口问问。”

顾南舟把铜铃拿起来，塞给他：“你话最多，手别停。”

马小满抱着铃，脸上写满悲壮：“我这个人，为队伍付出太多。”

他们出发时，离七点还有一个小时。

老街上的灯已经开始不正常。

有的灯没通电却亮着，有的灯亮了又灭，灭了又亮。有人站在自家门口，对着空巷子说话；有人抱着黑白照片哭；有人把门关得死死的，却从门缝里传出第二个人的笑声。

归人正在回家。

林照野经过周尧家楼下时，看见周尧妈妈坐在楼梯口，手里攥着寻人启事。她抬头看他们，眼睛肿得吓人。

“如果看见尧尧，”她说，“告诉他，妈在家等他。”

顾南舟的脚步停了一下。

林照野说：“我们会带真的回来。”

周尧妈妈看着他，像想问什么是真的，最后只是点了点头。

山神庙在雾岭半山腰。

庙很小，破得厉害，瓦片掉了一半，门口两只石狮子都没了头。庙前有一棵老槐树，树枝上挂满了红布条。风一吹，布条像一群没说出口的话。

六点四十分。

马小满开始摇铃。

叮铃。

叮铃。

声音很细，却压住了山里的雾。

庙门被林成川用那枚生锈钥匙打开。门后不是庙堂，而是一条往下的石阶。石阶两边嵌着小灯盏，灯盏里没有油，却一盏一盏自己亮起来。

林成川的脸色变得很难看。

“和十年前一样。”

顾南舟问：“我哥来过这里吗？”

赵砚秋说：“可能。”

“别可能了。”顾南舟说，“我今天要听真的。”

没人回答。

他们走下石阶。

铃声一直响。

叮铃。

叮铃。

石阶尽头是一条窄路。路两边没有墙，只有雾。雾里不时出现店铺、窗户、门牌、旧照片一样的人影。灯市街正在靠近现实。

沈星河的收音机忽然响起沈怀川的声音：

“星河，别来。”

她没有停。

又过了几步，林照野听见自己母亲在雾里叫他回家吃饭。

他咬住牙，没有回头。

马小满的铃声乱了一下。

雾里站着他外婆。

这一次，她没有拿冰袋，只拿着一只旧蒲扇。

“满伢子，”她说，“外婆不进门，就看看你。”

马小满眼泪哗地出来，手却仍在摇铃。

“你看吧。”他说，“我不回头，我正面给你看。”

老太太愣了一下，笑了。

然后她消失在雾里。

顾南舟走在最前面。

雾忽然散开一小块。

顾北川站在那里。

他比顾南舟记忆里高一点，穿着三年前失踪时的黑色汗衫，手里拿着一张戏票。

“南舟。”他说。

顾南舟停住。

林照野立刻抓住他胳膊。

顾北川看着他：“别灭灯。灯灭了，我就回不来了。”

顾南舟的眼睛红了。

“你是真的？”

顾北川笑了一下：“你小时候偷爷爷的刀练，割破手，哭得比谁都响。你不让我告诉别人。”

顾南舟浑身发抖。

这是真的。

只有他们兄弟知道。

沈星河轻声说：“它能学秘密。”

“不。”顾南舟看着哥哥，“这个没人说过。”

顾北川往前一步。

“南舟，带我回家。”

铃声忽然停了。

马小满惊叫：“我没停！”

铜铃裂了。

裂口从铃身中间开到边缘，像被什么东西咬了一口。

脚下的窄路开始移动。

林成川大喊：“跑！”

雾从两边扑过来，灯市街的门牌、供销社的柜台、青梧照相的橱窗同时贴近。林照野被人撞了一下，相机磕在胸口，疼得他差点喘不上气。

顾南舟却还站着。

顾北川伸出手。

“弟。”

顾南舟终于动了。

他没有去拉哥哥。

他把那张三年前的借条扔了过去。

“你要是真的，就知道你让我别去井边。”

顾北川脸上的笑停住。

顾南舟一字一句说：“你不会让我带你回家。你会让我走。”

雾里的顾北川沉默了。

然后他的脸开始模糊。

不是消失。

是像有人擦掉一张画，露出下面另一张空白的脸。

顾南舟转身就跑。

林照野第一次看见他哭。

他们冲出那条会移动的灯路时，山里已经完全黑了。

七点到了。

青梧县全城停电。

山神庙后的雾岭深处，千百盏灯同时亮起。

每一盏灯下面，都站着一个归人。

阿铃站在最前方，手里的灯笼终于亮了。

她看着他们，说：

“现在去灭真正的灯。”

林照野喘着气问：“在哪里？”

阿铃抬手，指向雾岭最高处。

那里有一座他们从未见过的牌楼。

牌楼下，两个周尧并肩站着。

一个白背心，一个校服。

他们同时抬头。

“你们来晚了。”
