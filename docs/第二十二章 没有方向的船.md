# 第二十二章 没有方向的船

那艘船没有船头。

林照野走上甲板后转了一圈，发现两端完全一样。两边都有船舱、桅杆和一盏熄灭的灯，连木板裂纹都对称得像镜子。

船中央放着一只铜罗盘。

指针不停旋转。

“好消息，”马小满说，“这次不是我们迷路。”

“坏消息？”顾南舟问。

“船也迷路。”

黑门在他们身后消失。

四周是没有边际的黑水。天上没有月亮，只有七颗星，位置和黑门图案一样。水面偶尔漂过门牌、照片、鞋子和没有写完的信，都是某个人走失时留下的东西。

船舱里有人。

一个短发女孩坐在桌边磨刀。

她十四岁左右，皮肤晒得很深，穿旧运动衫，裤脚卷到膝盖。看见五个人，她先抓起刀，又看见周尧校服上的青梧县小学字样。

“青梧？”

沈星河问：“你是谁？”

“江晚。白鹭镇。”

她没有放下刀。

“你们上船多久了？”

“刚上来。”

江晚看向窗外的七颗星。

“我三个月前上船。”

“现在是一九九八年九月。”沈星河说。

“我知道。白鹭镇才过去三天。”

船上的时间不按岸上走。

江晚在找弟弟江潮。三个月前，白鹭镇山洪冲走二十七人，九岁的江潮也在其中。江晚追着一盏水上航标进入北斗渡，此后到过三座渡口，却始终没找到弟弟。

“见过阿铃吗？”林照野问。

江晚从衣袋里拿出一小段红绳。

“她拿着一盏会指路的灯，被船带走了。”

沈星河问：“去了哪？”

江晚指向罗盘。

“它决定。”

五个人围住罗盘。

奇怪的是，每个人看见的指针方向都不同。

林照野看见它指向照相馆。

沈星河看见它指向县医院。

顾南舟看见它指向旧剧团。

马小满看见小卖部。

周尧面前的指针没有方向，只不停颤动。

江晚说：“船会先带你去最想去的岸。”

“那不是挺好？”马小满问。

“上岸容易，回来难。”

夜里，水面出现许多岸。

有的岸上站着亲人，有的是没发生事故的村庄，有的是已经长大后的自己。它们跟着船缓慢移动，只要有人说“靠岸”，船就会转向。

顾南舟看见顾北川站在剧团门口。

哥哥穿着戏服，冲他招手。

顾南舟闭上眼。

马小满看见外婆在小卖部里煮绿豆汤。他转身背对水面，嘴里不停背乘法表。

沈星河一直盯着船舱木板。

林照野问：“你看见什么？”

“我爸在修收音机。”

“想上岸吗？”

“想。”

她回答得很快，也很诚实。

“但想不是方向。”

江晚在一旁听见，冷笑了一声。

“你们青梧的人很会说放下。”

沈星河看向她。

“你不放？”

“如果一条路能让二十七个人活着，我为什么不选？”

“因为那条路可能会让另一些人消失。”

“你见过二十七双鞋从河里捞出来吗？”江晚问，“没见过，就别替他们讲道理。”

气氛一下子僵住。

马小满想插话，船突然震了一下。

第一颗星落到桅杆顶端。

罗盘停止旋转。

水面出现一条宽阔河岸，岸上亮着青梧县的灯。没有水库大坝，没有搬迁新村，旧县城完整地铺在山下。

码头牌子写着：青梧县老渡口。

林成川站在岸边。

他年轻、清醒，背着相机，笑着喊林照野回家吃饭。

沈怀川也在。

顾北川骑着自行车经过。

马小满外婆提着菜篮。

所有失去的人都在。

江晚看着他们：“第一渡口，未走之路。”

船靠岸了。

周尧第一个发现不对。

岸上所有人都在迎接他们。

却没有一个人认识他。
