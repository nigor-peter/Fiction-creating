# 第十三章 没有名字的人

“你终于把我的名字送来了。”

声音从水下那扇青门里传出来。

林照野站在船边，一只脚已经踩上船舷。

他不知道自己是什么时候站起来的。

那声音和他一模一样。不是灯市街里粗糙的模仿，也不是照片带来的回声。它甚至带着他说话前会轻轻吸气的习惯。

顾南舟一把将他拽回来。

“别听。”

林照野摔在船底，手肘撞得发麻。

水下青门猛地打开。

门后没有屋子，只有一片白光。一个无脸孩子站在门槛里，胸口贴着“林照野”三个字。它抬起手，像想隔着水面抓住船。

老吴用竹篙狠狠敲向水面。

水面发出玻璃碎裂一样的响声。

旧街的倒影向两边分开，修堤船猛地往下一沉。

马小满惨叫：“船漏了？”

“不是船沉。”老吴说，“是城升。”

水从四面八方漫上来，却没有打湿他们。雨夜、水库和远处堤坝像被一层透明布推到头顶。船底落在青石板路上，发出沉闷一响。

他们进入了水库旧城。

这里和灯市街不一样。

灯市街像一条被记忆拼错的县城倒影，店铺熟悉，门牌反着。水库旧城却是真正存在过的地方。石板上的裂纹、屋檐下的燕窝、戏院门口褪色的红柱，全有被时间磨过的痕迹。

只是没有人名。

刚才水面上那些写着名字的门牌，全变成了空白木牌。

老吴从船上下来，身体晃了一下。

“三更前找到万灯巷。”他说，“三更后，旧城会开始记你们。”

“记了会怎样？”周尧问。

老人看向他。

“你已经被另一个地方记过一次，还想试第二次？”

周尧不说话了。

沈星河的收音机忽然响起。

不是沈怀川的声音。

是一连串名字。

男声、女声、老人、孩子，重叠着从喇叭里念出来。有些清楚，有些只剩一个姓。每念一个名字，街边便有一盏灯亮起。

“把它关掉。”林照野说。

沈星河按下开关。

收音机没停。

她拔掉电池。

声音仍在继续。

“它不是在接收。”她说，“它在替这里点名。”

马小满立刻把收音机用衣服裹住：“那让它小声点名。”

他们沿旧街往前走。

老吴不肯带路，只坐在船头等。他说自己一旦踏进万灯巷，就会忘记为什么要回岸上。

街上渐渐出现人。

第一个是个穿灰布长衫的男人。他从药铺里走出来，脸上五官齐全，胸前却挂着一块空白木牌。

他挡住孩子们。

“请问，”男人很有礼貌地说，“我叫什么？”

没人回答。

男人又问一遍：“我叫什么？”

他的声音开始发抖。

沈星河说：“我们不知道。”

男人脸上的五官慢慢淡了。

“那我是谁？”

他伸手抓向沈星河。

顾南舟把她拉开。男人扑了个空，身体撞在墙上，像一张湿纸一样贴住，几秒后滑进墙缝。

更多人从门后出来。

挑担子的、抱孩子的、穿校服的、戴工帽的。他们来自不同年代，衣服不一样，问的却是同一句话。

“我叫什么？”

马小满开始回答。

“不知道。”

“不知道。”

“真不知道，叔叔您别逼我。”

每回答一次，那些人都会失望地退开。

林照野问：“为什么不随便给他们一个名字？”

马小满说：“我妈说名字就是门。乱给名字，不就等于把他们送进别人家？”

沈星河看了他一眼。

“你这次说得对。”

马小满挺了挺胸，又立刻缩回去：“但你别夸，我害怕有代价。”

他们在旧县衙门口找到第一条线索。

墙上贴着一张搬迁告示。纸已经烂了一半，下面盖着红章。所有住户姓名都被水泡掉，只剩住址。

万灯巷四号。

万灯巷九号。

万灯巷十七号。

最后一行用铅笔补写：灯戏班，无户籍，暂居旧戏台。

沈星河指着那一行：“阿铃。”

顾南舟说：“也可能是整个戏班。”

林照野举起相机拍照。

快门按下后，告示上慢慢浮出一只小手印。手印只有十一二岁孩子大小，食指上系着细红绳。

阿铃在这里留下过痕迹。

他们继续往万灯巷走。

周尧越来越慢。

马小满回头看他：“你怎么了？”

周尧靠在墙边，脸上全是冷汗。

“我想不起我妈叫什么了。”

“你妈就叫周尧妈妈。”马小满说完，自己先愣了一下，“不对，我也不知道她本名。”

周尧抬头，眼神惊恐。

“那个校服的在拿名字。”

街尽头出现一个穿校服的背影。

袖口少了一颗扣子。

他没有回头，只沿着万灯巷慢慢走。每经过一扇门，空白门牌上便多出一个名字。

马小满。

沈星河。

顾南舟。

周尧。

最后一块门牌上，水痕缓缓组成三个字。

林照野。

“追！”顾南舟冲出去。

他们跟着校服周尧穿过万灯巷。两边的无名者越来越多，却没有再问名字。他们全都看着那道校服背影，像饿了很久的人闻见饭香。

校服周尧跑进旧戏台。

戏台幕布已经烂掉，台下整齐摆着数十把空椅。每把椅背都挂着一块无名牌。舞台中央放着一盏没有点燃的青铜灯。

第二盏灯。

赵砚秋不在。

灯旁只放着一本摊开的县志残卷。

残卷上密密麻麻写满名字，最后一页却只有两个字。

阿铃。

一个无名人站在灯后。

看身形，是个十二三岁的孩子。没有五官，没有头发，身上穿着分不出年代的灰衣。它伸出手，在灰尘里写下“阿铃”两个字。

沈星河问：“你认识她？”

无名人摇头。

“那你为什么写她的名字？”

它抬手指向自己的胸口。

那里有一道被撕掉木牌留下的伤痕。

它张开嘴，发出的却是许多人的声音：

“因为这是我们最后记得的名字。”

戏台下，几十把空椅同时轻轻转向他们。

每一块无名牌上，都浮出同样两个字。

阿铃。

沈星河的收音机突然停止点名。

短暂的安静之后，里面传出赵砚秋的声音。

“你们不该来得这么快。”

舞台上那盏青铜灯，亮了。
