# 第二十章 万灯归处

水库旧城升出水面的那一刻，青梧县所有人都看见了。

老街尽头多出一座牌楼。

县医院走廊通向万灯巷。

照相馆暗房的门打开，门后是沉城戏台。

水库、灯市街和现实青梧三层重叠，每家门前都亮起一盏灯。门牌上的姓名化成水迹，沿墙往下流。

人们站在街上，开始忘记彼此。

丈夫叫不出妻子，母亲认不出孩子，老师望着花名册不知道该喊谁。只有那些已经失去的人，从灯下变得越来越清楚。

送灯从沉城戏台出发。

阿铃提灯走在最前。

顾南舟按照走边步法开路。每一步踩下，现实与旧城之间便出现一块稳固石板。

马小满抱着点名册，边走边念。

“魏家老二，记在魏老太太床头那张照片里。”

“县医院护工，记在旧楼值班表里。”

“马家外婆……”

他的声音停了一下。

外婆站在小卖部门口，手里拿着橘子冰袋。

这一次，她有影子。

“满伢子。”

马小满眼泪掉下来，继续念：

“马家外婆，记在我妈做酸米饭会多放一勺糖，记在柜台下那罐话梅，记在我身上。”

外婆笑着点头。

她没有要求进门。

灯光一晃，她化成一张普通旧照片，落在小卖部柜台上。

沈星河背着拆开的收音机。扬声器连接现实县广播，沈岚、林成川、马莲、顾怀声和全县还记得名字的人，一起跟着马小满回应。

每一个名字被正确回应，门牌便重新固定在现实的房子上。

每一个没人记得的名字，则被阿铃送进灯笼里的路。

顾北川走到路口。

顾南舟的步法停了一拍。

“哥。”

顾北川笑：“这次可以叫。”

“为什么？”

“因为你不是叫我回来。”

顾南舟点头。

他完成最后一个武生亮相，木刀横在胸前。

“顾北川，送灯。”

顾北川走进灯路。

没有回头。

沈怀川也出现了。

沈星河的收音机已经听不见他的声音。她只能隔着灯光看父亲。

沈怀川嘴唇动了动。

沈星河读出来：长大了。

她没有说留下。

“沈怀川，送灯。”

父亲向她点头，跟着顾北川走进光里。

另一个林照野站在青梧照相门口。

他已经给自己选了名字。

木牌上写着：照影。

“你不怕没人记得你？”林照野问。

照影举起相机：“你拍我。”

林照野下意识抬起自己的相机。

镜头里，照影看起来和他完全一样。他只要按下快门，这个新名字就会在现实留下第一份证明。

可他忽然想起自己一直把相机当成抓住真相的手。

有些东西被拍下，就像被占有。

“我不拍。”林照野说。

照影脸色变了。

“为什么？”

“因为你不是为了给我证明才存在。”

林照野第一次主动把相机放下。

“你自己走。”

照影看了他很久。

然后笑了一下，提着自己的相机走进灯路。

赵砚秋牵着雨灯，走在最后。

水库旧城开始崩塌。

第二盏灯的火越来越弱，第一盏灯芯也烧到尽头。阿铃的身体随着灯光变淡，她开始忘记每个人的称呼。

“照相的。”

“听匣子的。”

“怕鬼的。”

“唱武戏的。”

她又回到了最初认他们的方式。

马小满哭着说：“我叫马小满。你给我记住。”

阿铃认真重复：“马小满。”

“她叫沈星河。”

“沈星河。”

“他叫顾南舟。”

“顾南舟。”

“他叫林照野。”

阿铃看向林照野。

“林照野。”

四个人一起叫她：

“阿铃。”

灯路震动。

几百年来，她第一次不是因为无名者需要一个送灯人而存在，而是因为四个朋友记得她。

阿铃胸前出现一块小小木牌。

她笑了。

“这次我有名字了。”

送灯走到万灯巷尽头。

阿铃把两盏灯合成的灯笼放进水中。灯没有熄灭，而是化成无数细小光点，分别流向每一个该去的位置。

有的进入照片。

有的进入名字。

有的沿着一条没人知道终点的路远去。

第二盏灯熄灭。

旧城沉回水底。

现实青梧的门牌重新出现，医院病历、学校花名册和户籍记录恢复。所有人像从一个漫长梦里醒来，重新叫出身边人的名字。

赵砚秋没有回岸上。

有人说最后看见他站在旧戏台前，目送雨灯走进灯路。也有人说他跟着船回来了，只是忘记了自己是谁。

阿铃也不见了。

暑假结束前一天，林照野在照相馆整理底片。

他没有拍送灯最后一刻。

可暗房里仍然多出一张照片。

照片上是沉回水底的万灯巷。

巷子尽头站着阿铃。她穿青布褂，胸前挂着自己的名字，手里没有灯笼。她身后有一扇从未见过的黑门。

门上画着七颗星。

门牌写着三个字：

北斗渡。

照片背面没有水字。

只有阿铃歪歪扭扭写下的一句话：

灯路不是只有一条。

林照野把照片拿出暗房。

沈星河、马小满、顾南舟和周尧正在外面等他。开学报名表摊在柜台上，每个人都已经写好自己的名字。

马小满问：“又拍到什么不该拍的了？”

林照野看着照片，过了一会儿，把它放在桌上。

“一扇门。”

顾南舟问：“开吗？”

窗外，青梧县小学的开学铃响了。

林照野笑了一下。

“先报名。”

照片里的黑门后，传来极轻的一声船铃。

叮。
