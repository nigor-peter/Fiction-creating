# 《雾岭灯火》故事圣经

## 基本定位

类型：少年探险、民俗悬疑、超自然冒险。

时代：1998 年夏天。

地点：西南山区青梧县。县城因水库搬迁逐渐衰落，旧街、供销社、录像厅、废弃电影院、县医院、文化馆和雾岭山路构成主要舞台。

创作原则：保持原创。可借鉴群像冒险、少年视角、双线调查、怀旧小镇、现实危机与超自然并行等通用写作方法，但不复刻任何现成作品的人物、场景、怪物、对白或标志性桥段。

## 核心设定

青梧县外有一片常年起雾的山岭，名为雾岭。当地传说雾岭深处有一座沉入地下的旧县城，城中有一盏“引魂灯”。农历七月，山里会传来灯戏唱腔和皮影锣鼓。听见的人若回头，可能看见已经失踪或死去的人站在雾里。

真相不是单纯的鬼怪。青梧县地下存在一个与现实重叠的“地下青梧”：它像现实县城的倒影，保存着被遗忘的人、旧物、愿望和恐惧。古人以引魂灯维持边界；现代地质实验破坏了原有平衡，使地下青梧开始学习、模仿并进入现实。

## 主题

1. 失去的人是否真的能回来。
2. 小地方的秘密如何一代代传下去。
3. 少年友情如何抵抗成人世界的隐瞒。
4. 记忆既能救人，也能诱骗人。
5. 真相不总能修复伤口，但能让人停止被谎言控制。

## 标志性意象

引魂灯、雾岭山路、反向门牌、湿漉漉的旧街、皮影幕布、县志残页、黑白照片、BP 机、录像厅雪花屏、搪瓷杯、旧防空洞、山神庙、水库淹没线。

## 悬疑规则

地下青梧不能凭空创造完整的人，只能模仿被强烈记住的人。

影人越被亲近之人相信，就越接近真实。

引魂灯不是召魂工具，而是边界稳定器。灯亮太久，两个世界会贴近；灯灭太久，被困者无法找到出口。

照片、录音、皮影和镜面能留下两个世界重叠时的痕迹。

雾不是入口本身，而是边界变薄时的自然现象。

## 叙事结构

少年线负责行动、发现和情感推进。成人线负责旧案、实验、地方志和历史债务。两条线应在每个关键节点交叉，但不要让成人完全替孩子解决问题。

每章最好包含：

1. 一个现实问题。
2. 一个超自然异常。
3. 一个关系推进。
4. 一个具体悬念物：照片、唱词、录音、灯、门牌、信件、失踪物。

## 第一季主线

暑假第一天，周尧在雾岭附近失踪。林照野、沈星河、马小满、顾南舟追查同伴下落，发现旧电影院后方的防空洞与一张不存在的青梧县地图。他们误入地下青梧，见到两个周尧，并逐渐发现十年前的地磁实验与更古老的引魂灯祭祀有关。

七月十五夜，青梧县全城停电。雾岭亮起成排灯火，许多失踪多年的人回到街口。孩子们必须判断哪些人是真正的幸存者，哪些只是地下青梧借记忆造出的影子。

季终：引魂灯被熄灭，裂隙暂时关闭。周尧获救，阿铃消失。林照野冲洗照片时，看见沈星河父亲站在不该存在的清代牌楼下，手里举着刚点燃的灯。
