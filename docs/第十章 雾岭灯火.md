# 第十章 雾岭灯火

周尧回来了。

这是青梧县第二天最大的消息。

比昨晚全城停电、雾岭灯火、山神庙塌了一半都大。

大人们更愿意相信一个简单的说法：周尧在山里迷路，被几个同学和大人一起找到了。他饿了三天，冷了三天，所以说不清自己去了哪里，也记不清发生了什么。

派出所的人来问话时，周尧坐在病床上，手里捧着一碗粥。

刘警官问：“你这几天在哪？”

周尧想了很久。

“街上。”

“什么街？”

周尧皱起眉。

“很多灯的街。”

病房里安静了一下。

刘警官把笔放下，轻声说：“孩子受惊了，慢慢养。”

周尧妈妈坐在床边，紧紧抓着儿子的手。她没有再问哪个是真的，也没有问为什么儿子袖口少了一颗扣子。她只是看着周尧吃粥，像只要他还会喊饿，世界就没有完全坏掉。

林照野他们站在病房外。

顾南舟隔着门看了很久。

“他忘了。”他说。

沈星河说：“也许这样好。”

“我哥没回来。”

没人接话。

顾南舟低头看着手里的借条。昨晚从雾里回来后，借条上的字淡了很多，顾北川的名字只剩一个模糊的影子。

“但他让我走。”顾南舟说。

这句话他说得很轻。

像终于承认那是哥哥留给他的最后一句话。

马小满手里拿着两根冰袋，一根酸梅，一根橘子。

他把橘子味那根放在医院窗台上。

林照野看着他。

马小满吸吸鼻子：“我不吃。放着。”

“给谁？”

“不知道。”马小满说，“可能给外婆，也可能给我自己。反正今天不想吃。”

窗外的槐树叶被风吹动。

那根橘子冰袋安静地躺在阳光里，包装上没有奇怪日期，红绳也没有反着打结。它只是夏天里一根普通冰袋。

这反而让马小满哭了。

沈岚没有再锁沈星河的门。

她在医院旧楼值完夜班，回家后坐在厨房里，一直坐到天亮。沈星河进门时，她把一碗面推到桌上。

母女俩谁都没有先说话。

最后是沈岚开口：

“你爸那段录音，我想再听一次。”

沈星河把收音机放在桌上。

收音机已经坏了。

昨晚从山神庙回来后，它再也收不到第七号井的频率。无论怎么调，里面都只有普通县广播，天气预报，寻人启事取消通知，还有卖农机配件的广告。

沈星河说：“听不了了。”

沈岚低头。

过了很久，她说：“那就算了。”

沈星河看着母亲鬓边的白发，忽然说：“他让我不要找他。”

沈岚的手停住。

沈星河继续说：“但他说的是不要找他，不是不要记得他。”

沈岚的眼泪掉进碗里。

那天上午，林成川重新打开照相馆。

他把橱窗里的旧照片一张张取下来，擦干净，又摆回去。林照野站在旁边帮忙。父子俩很少说话，但这一次，沉默不像以前那样沉。

林成川把那卷十年前的底片递给他。

“以后你保管。”

林照野接过来：“你不怕我乱洗？”

“怕。”林成川说，“但我更怕你什么都不知道。”

林照野愣了一下。

林成川看着橱窗玻璃里的自己。

“我从井里出来以后，一直以为不说就是保护你。后来发现，不说只是把你一个人推到更黑的地方。”

林照野低头看底片。

“爸。”

“嗯。”

“你在灯里说，你不完整。”

林成川的手停了一下。

“那不是我。”

“我知道。”林照野说，“但它学得很像。”

林成川沉默。

过了一会儿，他说：“因为我确实这么想过。”

林照野抬头。

林成川没有躲。

“人回来以后，不代表什么都回来了。可是照野，缺了一块，也能活。”

林照野看着父亲。

这是他第一次觉得，林成川不是那个被雾岭吓坏的大人，也不是暗房里反复擦照片的影子。

他只是一个活着回来的人。

黄昏时，四个孩子去了雾岭脚下。

山神庙被封了，门口拉着警戒线。官方说是年久失修、昨晚山体滑坡。旧槐树还在，红布条被雨打湿，垂在枝头。

阿铃不见了。

她没有留下灯笼，也没有留下脚印。

只有庙门口石阶上，有一小段烧黑的灯芯。

沈星河捡起来，放进一个空胶卷盒里。

“她死了吗？”马小满问。

没人回答。

顾南舟说：“她说自己不是来回家的。”

林照野看向雾岭深处。

昨晚那些灯都灭了。山还是山，树还是树，雾也淡得像普通水汽。如果不是周尧真的回来了，如果不是他们每个人口袋里都多了一些不该存在的东西，林照野几乎会以为那只是几场连在一起的噩梦。

马小满忽然说：“你们有没有觉得，少了点什么？”

沈星河问：“什么？”

“以前我觉得暑假很长。”马小满说，“现在感觉作业都亲切了。”

顾南舟看他：“你可以替我写。”

“亲切归亲切，别乱认亲。”

林照野笑了一下。

这是周尧失踪后，他第一次真的笑出来。

回到照相馆后，他洗了最后一卷胶片。

那是昨晚在引魂灯前拍的。

林成川说不要一个人洗，所以沈星河、顾南舟、马小满都挤在暗房外面等。马小满坚决不进暗房，说红灯对他的命格不友好。

第一张照片显出来，是山神庙。

第二张，是灯路。

第三张，是祭台。

第四张，是阿铃拔下灯芯的瞬间。

照片里，她站在火光中，青布褂被照得透明，手里的灯笼亮着。她没有看引魂灯，而是看向镜头。

像早就知道林照野会拍这一张。

第五张照片慢慢显影时，暗房里的温度降了一点。

林照野屏住呼吸。

照片上不是雾岭。

也不是灯市街。

是一座更旧的牌楼。

牌楼上写着两个字：

青梧。

下面站着三个人。

阿铃站在中间，手里提着灯笼。

左边是沈怀川。

右边是顾北川。

沈星河看到父亲时，没有哭。她只是把手按在暗房门框上，指节发白。

顾南舟也没有说话。

照片里的顾北川比他们记忆里大了一点，像在那边又长过几年。他看着镜头，手里拿着一张戏票。戏票背面写着一行字：

别让下一盏灯亮。

林照野还没来得及说话，照片角落又浮出一个人影。

那个人站得很远，几乎被牌楼阴影盖住。

可林照野认出来了。

是赵砚秋。

年轻的赵砚秋。

他手里拿着的不是唱本。

是一盏已经点燃的灯。

暗房外，县广播忽然响起晚间报时。

“北京时间，七点整。”

林照野低头看照片。

照片背面慢慢渗出水痕，浮出一行小字：

第一盏灯已灭。

第二盏灯将醒。

马小满在门外小声问：“我能不能现在开始怀念作业？”

没有人笑。

因为暗房红灯闪了一下。

随后，整个青梧县的电灯同时亮了又灭。

只一瞬。

像有人在很远的地方，眨了一下眼。
