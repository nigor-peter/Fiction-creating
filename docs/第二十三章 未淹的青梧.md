# 第二十三章 未淹的青梧

没有修建水库的青梧县，比他们的县城大一倍。

老街一直延伸到河边，青石板干净，店铺没有空，旧电影院还在放电影。山神庙香火旺盛，雾岭只是普通山岭。没人听说灯市街，也没有第七号实验井。

林成川一把抱住林照野。

“跑哪去了？你妈找你半天。”

父亲的怀抱温暖，身上有显影液和肥皂味。他说话清楚，眼睛里没有红灯留下的阴影。

林照野僵着手，没有抱回去。

他等这一刻太久了。

可真正等到时，他先想到现实里的父亲。那个承认自己缺了一块、仍然重新打开照相馆的人。

眼前的林成川很完整。

也很陌生。

沈星河在县医院门口见到沈怀川。

父亲比照片里老一些，白衬衫袖子卷着，手里拿着一袋中药。他自然地摸了摸她的头，问她收音机修好没有。

沈星河低头，眼泪落在鞋面上。

顾北川从剧团出来，拍了顾南舟后脑勺一下。

“发什么呆？爷爷等着排戏。”

马小满外婆骂他又胖了，把一根橘子冰袋塞进他手里。

这里不是粗糙的幻觉。

每个人都有自己的生活，时间完整，记忆连贯。沈怀川记得沈星河出生那天，顾北川知道弟弟藏木刀的地方，外婆左手红绳系得正确。

江晚站在码头，冷冷地说：“我告诉过你们。这是真正可能发生的路。”

周尧一个人站在人群外。

他去了学校。

花名册没有周尧。教室里他的座位坐着另一个孩子。老师不认识他，粮站后面的周家房子也不存在。

没有水库搬迁，周尧父母就没有来到青梧。

“那我去哪了？”周尧问。

没人能回答。

江晚说：“也许在别的县，也许没出生。”

马小满立刻把冰袋扔到地上。

“那我们走。”

外婆在后面喊他。

他没有回头。

“我好不容易才认对一次，不能为了再见她，把周尧认没了。”

周尧低着头：“你们可以留下。”

顾南舟问：“然后把你一个人扔船上？”

“这里你哥活着。”

“我哥已经做过自己的选择。”

周尧看向沈星河。

沈怀川站在医院门口，眼里都是担心。

“星河，”他说，“你不是一直想知道，如果我没去实验井，我们会怎样吗？”

沈星河走到他面前。

“会怎样？”

“我陪你长大。教你修更好的机器。你妈不会一个人熬夜。”

“代价呢？”

沈怀川沉默了一下。

“周尧不在这里。”

“只有周尧？”

“还有一些因水库搬迁才来到青梧的人。”

“多少？”

“四百三十七人。”

沈星河点点头。

“那这条路不是我的。”

沈怀川眼里露出痛苦。

“我是真的。”

“我知道。”

这比面对影子更难。

假的东西可以拒绝。真正可能拥有的幸福，才需要选择。

沈星河抱了父亲一下。

“但你属于没有发生的那条路。”

她松开手，转身走向码头。

林照野最后经过照相馆。

这个世界的林成川追出来，把一张全家福塞给他。照片里父亲、母亲和林照野都笑得自然。

“带着。”父亲说，“忘了路就看。”

林照野接过照片。

走到船边时，他把照片放在码头石阶上。

不是因为它不真。

正因为它真，他不能把它偷到另一个现实里，当成现实父亲永远不够完整的证据。

五个人重新上船。

江晚没有动。

“你们宁愿保留发生过的灾难？”

周尧说：“我不是灾难的附带品。”

江晚看着他。

周尧继续说：“你想救弟弟，可以。但别先把别人算成可以没有。”

船离岸。

未淹青梧的灯火渐渐远去。

沈怀川一直站在码头，直到看不见。

船舱桌上多出一张未寄出的明信片。

正面是白鹭镇。

背面写着：

江晚，如果我没去庙会，你会不会变成另一个人？

落款是江潮。

江晚抓起明信片，脸色第一次变了。

水面上传来船桨声。

一个戴斗笠的男人撑着小船靠近。

“想知道答案，”他说，“先买船票。”
