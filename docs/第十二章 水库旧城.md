# 第十二章 水库旧城

青梧水库管理所里挂着两张地图。

一张是现在的。

蓝色水面占了大半张纸，坝体、泄洪道、管理所、移民新村都标得清清楚楚。

另一张被玻璃压在下面，纸已经发黄。那是水库修建前的青梧县。老街、粮站、县衙、祠堂、戏台和一条叫万灯巷的窄街，都沉在如今的水面下面。

四个孩子站在地图前，谁也没说话。

马小满看了很久，指着水底一处：“我家以前在这里。”

“你家不是一直在老街？”周尧问。

马小满瞪他：“你记得我家，还是不记得我名字？”

周尧张了张嘴。

“马……”

后面的字仍然卡住。

马小满摆摆手，装作不在意：“算了，反正你以前也老欠账。”

水库管理员是个晒得黝黑的中年人，姓罗。他不让孩子们碰旧地图，更不肯借船。

“这几天下雨，水位涨得快。你们几个小孩来水库做什么？”

沈星河把文化馆借阅卡推过去：“赵砚秋来过吗？”

罗管理员看见灯形印章，脸色微微一变。

“没见过。”

“那你为什么认识这个印？”顾南舟问。

“我不认识。”

顾南舟还要追问，马小满忽然走到墙角一张搬迁合影前。

照片里站着几十户人家。大家胸前都挂着白布条，写着姓名和原住址。第一排有个抱着算盘的年轻女人，眉眼和马小满妈妈很像。

她胸前写着：万灯巷四号，马莲。

“这是我妈。”马小满说。

照片角落还有一个老人。

别人都面向镜头，只有老人背对大家，看着远处旧戏台。他胸前的白布条没有名字，只写着四个字：拒绝搬迁。

罗管理员走过来，把照片翻扣在桌上。

“别看了。”

林照野问：“他是谁？”

“一个疯老头。”

“名字呢？”

罗管理员没有回答。

门外忽然传来竹篙敲地的声音。

笃。

笃。

一个瘦得像枯树根的老人站在管理所门口。他戴斗笠，披蓑衣，左眼蒙着白翳，手里拄着长竹篙。

“名字是你们淹掉的。”老人说，“现在倒来问我？”

罗管理员脸色难看：“老吴，你又来闹什么？”

老人没理他，只看着林照野。

“成川家的孩子？”

林照野点头。

老人伸出手：“照片给我。”

林照野没有马上动。

老人说：“你胸口的名字快被人拿走了，还舍不得一张纸？”

这句话让所有人安静下来。

林照野拿出无脸孩子的照片。

老人只看一眼，就把照片放到鼻子前闻了闻。

“水腥。”他说，“第二盏灯在醒。”

罗管理员突然站起来：“这里没有什么灯。你们都出去。”

老人转头看他：“你夜里听见万灯巷敲门，不敢开，就当城没沉过？”

罗管理员的脸一下子白了。

老人姓吴，没人知道全名。搬迁时他拒绝离开旧城，后来被人抬上岸。几十年里，他一直住在水库边，靠修船、补网和给管理所看雨尺过日子。

他把孩子们带到堤坝后面。

芦苇丛里藏着一艘窄窄的修堤船，船身刷过黑漆，边缘挂着铜铃。每只铃铛里面都塞了棉花，风吹也不会响。

“今晚无月。”老吴说，“雨到三更，旧城会浮上来。”

马小满看向水面：“真的浮？房子从水里冒出来？”

“不是房子浮。”老人说，“是你们沉下去。”

马小满立刻往后退：“这区别一点也没安慰到我。”

沈星河问：“赵砚秋是不是坐过这条船？”

老人沉默了一会儿。

“七天前。”

“他去了哪里？”

“万灯巷。”

“为什么让他去？”顾南舟问。

老人用竹篙敲了敲船舷。

“我没让。他叫出了我的名字。”

孩子们都看向他。

“你叫什么？”马小满问。

老人抬起那只白翳眼睛。

“忘了。”

他说这两个字时，没有悲伤，只有一种被水泡久了的平静。

“第二盏灯先拿走名字，再拿走别人记得你的地方。赵砚秋带着县志来，是想把旧名字一个个叫回来。”

沈星河问：“叫回来会怎样？”

老人看向林照野。

“活人让路。”

傍晚，马小满把旧地图带回家给母亲看。

马莲刚看见万灯巷三个字，手里的菜刀就停了。

“谁给你的？”

“水库管理所。”

“不准去。”

“妈，你认识老吴吗？”

马莲的脸色变了。

“什么老吴？”

“搬迁合影里那个没名字的老人。”

马莲把菜刀放下，关上厨房门。

她从米缸底下摸出一只布包。里面是一把旧铜钥匙和一块白布条。

白布条上写着：万灯巷四号，马莲。

“搬迁前一晚，”她说，“万灯巷家家户户在门口点灯。老人说，房子可以淹，名字不能淹。第二天上船时，老吴给每家一把钥匙，说以后旧城找不到门，就用名字开。”

她把钥匙递给马小满。

“我一直没扔。”

马小满第一次没有开玩笑。

“妈，我们得下去。”

马莲看着儿子很久。

“你外婆回来那次，我差点认了她。”她说，“这次你要去，我拦不住。但你记住，水面出现旧街的时候，不要叫出失去的人的名字。”

“为什么？”

“水底没有门牌。”

她握住马小满的手，把钥匙压进他掌心。

“名字就是门。”

午夜前，五个孩子到了水库边。

周尧坚持跟来。他说那个校服周尧每晚都在梦里往水库走，他们如果不带他，另一个他也会来。

老吴已经在船上等着。

雨从九点开始下。没有风，雨丝笔直落进水库，水面密密麻麻全是针眼。修堤船没有发动机，老人只用竹篙慢慢撑。

岸上的灯越来越远。

沈星河把收音机放在膝上。里面没有广播，只有水声。林照野的相机用黑布包着。顾南舟握着船边的绳。马小满把母亲给的铜钥匙穿在脖子上，不停摸。

船行到水库中央时，老吴停下竹篙。

“来了。”

雨声忽然变远。

水面下亮起第一盏灯。

接着是第二盏、第三盏。

一条街从黑水里显出来。瓦檐、木窗、牌楼、戏台，都倒映在水面，却不像倒影，因为所有灯火都是正着燃烧。

修堤船像悬在街道上空。

马小满低头，看见船下是一家小卖部。门口没有号码，只挂着一块木牌。

马莲。

再往前，每一扇门上都写着一个人名。

有活人的。

也有死人的。

沈星河忽然抓住林照野的手臂。

水面正下方，有一扇青色木门。

门牌上写着：

林照野。

门从里面轻轻敲了三下。

笃。

笃。

笃。

一个和林照野一模一样的声音说：

“你终于把我的名字送来了。”
