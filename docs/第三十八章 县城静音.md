# 第三十八章 县城静音

无声林在他们离开后开始扩张。

最先静下来的，是县医院。

护士按呼叫铃，红灯亮了，铃却不响。病人张嘴喊疼，只有喉咙上的筋在动。沈岚站在走廊里，听见自己心跳一下一下撞着胸口，却听不见脚步声。

接着是照相馆。

林成川按下快门，快门没有响。暗房里的定时器走到零，铃声消失。照片显影时，水面上浮出一片灰绿色树影。

然后是小卖部。

算盘珠子被马莲拨得飞快，却没有噼啪声。玻璃柜里的糖纸一张张褪色，墙上的赊账本自动翻页，空白从边角往中间爬。

最后是学校。

下午第一节课，老师在黑板上写字，粉笔无声地断成两截。全班孩子看着断粉笔落地，没有一个人说话。

不是不想说。

是说不出来。

县城像被整块棉被盖住。

广播站的喇叭全部失声，汽车喇叭失声，菜市场吆喝失声，河边洗衣服的木槌失声。连雾岭方向修路的伐木声都没有了，只剩地图西北角那块灰绿色林区继续扩大。

林照野站在照相馆门口，第一次没有举相机。

他忽然明白，相机救不了这件事。

照片能证明某个瞬间存在，却不能替所有人记住某个人为什么值得存在。

他跑到小卖部，把马小满家的赊账本抱出来。

马小满也明白了。他没有问，直接把柜台下面所有旧票据、糖纸、欠条、进货单装进书包。

顾南舟去剧团搬戏谱和旧锣。

周尧回学校拿花名册。

沈星河带着沈岚去医院档案室，把旧病历、接生登记、输液单和死亡证明一捆捆搬出来。

阿铃站在街口。

她胸前的姓名牌一半写着“岑小铃”，一半写着“阿铃”。两个名字像在同一块白塑料上互相争位置。

“去广场。”她说。

声音很小，却传了出去。

这是静音后的青梧县里，第一句真正被听见的话。

傍晚，县政府前的小广场堆满了东西。

账本、照片、戏票、病历、借条、户口簿、旧奖状、墓碑拓片、结婚登记、卖粮票据、车票、作业本。很多东西看起来不体面，甚至有点可笑。

可它们都是人活过的痕迹。

赵砚秋抱来县志。

他把那本漂亮的精装县志放在最上面，又沉默地把它拿下来，放到底下。

“县志不该压在这些东西上面。”他说。

马小满看了他一眼：“你终于说了句人话。”

赵砚秋没有反驳。

广场边缘陆续来了很多人。

起初他们只是看。看见自己的老账、旧照片、病历和签名，才慢慢走近。有人认出一张借条上的字，有人认出一件旧戏衣上的补丁，有人说自己小时候听老人提过西北沟采木队。

无声没有完全退去。

每个人说话都像隔着厚墙。

林照野把一块木板立起来，在上面写：

无声县志。

写完最后一笔，木板上的字忽然变白。

标题消失了。

所有证据堆成一座小山，第一页却仍然没有名字。

何栖站在人群外。

他问：

“你们要写谁？”

没人立刻回答。

因为大家终于意识到，这不是写一群“可怜的失踪者”。

这是把青梧欠下的第一笔账，写回现实。
