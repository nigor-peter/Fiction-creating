# 第三十三章 林场旧证

废弃林场在县城西北，过了老水泥厂，再往雾岭方向骑半个小时。

路是新铲出来的，黄泥翻在两边，像刚被剥开的伤口。路边插着几根木桩，上面刷白漆，写着“林区开发，闲人勿入”。

马小满骑得最慢。

“我先声明，我不是怕。”他喘着气说，“我是觉得闲人勿入这四个字通常很有道理。”

顾南舟扛着一根木棍走在前面：“你要是怕，就回去看店。”

“我怕归怕，回去归回去，这是两码事。”

阿铃没有笑。她从离开县城开始就一直回头看，像有什么东西跟在他们后面，又像有什么东西被他们落在了后面。

林场大门只剩半扇。铁牌上“青梧县雾岭林场”几个字锈得发黑。门卫室窗户破了，里面有一张桌子、一把断腿椅和半缸雨水。

沈星河先进去。

桌上压着一本护林登记册。纸页发霉，边角粘在一起。她小心揭开，前面全是正常记录：巡山、封火、禁猎、木材调运。

翻到一九九一年七月，字迹突然乱了。

七月十三，西北沟听见树响，无风。

七月十四，三号伐区罗盘失灵，工人少一人，点名不少。

七月十五，停工。夜里有人敲树，三长三短。

七月十六，陈守根请示封沟。

七月十七，记录页被墨水涂黑。

林照野用手指摸过那片黑。墨水下似乎还有字，但看不清。

顾南舟在柜子里找到一沓伐木证。最上面几张盖着县林业站的红章，下面却夹着一种没有编号的旧证。

证件名称是：灯芯木采伐临时通行证。

采伐单位：青梧县第零生产队。

有效区域：无声沟。

马小满咽了口唾沫：“第零生产队真有啊。”

“为什么叫第零？”周尧问。

没人回答。

他们又在墙角铁皮箱里找到一张照片。

照片被油布包着，保存得比其他文件好。画面里是林场空地，二十多个人站成两排，手里拿锯、斧头和麻绳。最前面有个小女孩，胸前挂着铜铃。

女孩的脸是白的。

不是曝光过度，而是整张脸像被人用橡皮擦掉。

照片背面写着一排名字。

林照野念出第一个：“何……”

他的声音断了。

不是卡壳。

是断。

他明明能感觉喉咙震动，空气却没有带出声音。嘴唇继续动，第二个字消失在口腔里。

沈星河一把捂住照片背面：“别念！”

林照野扶着桌子，半分钟后才重新吸到自己的声音。

“何什么？”顾南舟问。

林照野摇头。他看见了字，却像看见一条鱼从水里翻身，又立刻沉下去。

门外传来脚步声。

一个老人站在林场大门口，背着柴刀，穿一件洗得发白的蓝布褂。他的眼睛很浑，却一下就看见了桌上的照片。

“谁让你们翻这个的？”

顾南舟握紧木棍。

老人没有进来，只盯着照片：“我叫陈守根，以前看这片林子。你们要是不想以后说话只剩嘴动，就把东西放回去。”

沈星河问：“无声沟在哪？”

陈守根脸上的皱纹抖了一下。

“没有这个地方。”

“那第零生产队呢？”

老人沉默很久。

风从门卫室破窗钻进来，吹得登记册哗啦啦翻页。翻到最后一页时，纸上只有一句话：

他们不是失踪。

下面还有半句，被水泡模糊了。

陈守根替他们念了出来。

“因为没人找。”

林场外，山里忽然传来一阵砍树声。

咚。

那声音沉得不像斧头砍在木头上。

倒像有人从树心里敲门。
