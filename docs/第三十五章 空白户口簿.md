# 第三十五章 空白户口簿

旧户籍室在派出所后院。

那是一排低矮平房，墙上刷着早就褪色的标语：户口登记，利国利民。门锁锈死，窗户里堆满纸箱和旧档案袋。

沈岚和赵砚秋是在天黑后来的。

沈岚穿着白大褂外套，手里拿着一串钥匙。赵砚秋抱着一只牛皮纸箱，箱子上写着“地方志补遗”，四个字被雨泡得发胀。

“你们不该把孩子带到这里。”沈岚说。

林照野反问：“那谁该来？”

沈岚没有回答。

赵砚秋把钥匙插进锁孔，转了两次没转动。顾南舟上前用木棍一撬，锁“咔”地开了。

“文物保护意识很差。”赵砚秋说。

顾南舟：“下次你慢慢保护，我们先被林子吃了。”

户籍室里全是霉味。木架上放着一排排蓝皮簿子，按年份、乡镇、公社分开。沈星河打开手电，光柱里全是灰尘，像一场倒着下的雪。

赵砚秋直奔最里面的柜子。

“第零生产队不会在正式户籍里。”他说，“如果有，只能在临时迁入、劳力调配或者事故补登里。”

“你早知道？”周尧问。

赵砚秋的手停了一下。

“我知道有空白，但不知道空白里有人。”

这句话让屋里安静下来。

他们找到一本一九六九年的旧簿。封面没有标题，只贴着半张红纸，红纸上写了一个“零”。

沈岚翻开第一页。

纸上密密麻麻全是登记格，却没有姓名。出生年月、籍贯、家庭关系、迁入原因，全都空着。只有少数格子边缘留着被钢笔划破的痕迹。

马小满把小卖部赊账本从怀里掏出来。

“看这个。”

他把账本摊在户口簿旁边。那些压痕、糖纸、欠条和旧货单看起来乱七八糟，比正式档案寒酸得多。

可它们有生活。

盐二斤。火柴一盒。红糖半斤。煤油三两。草鞋一双。针线一包。

“我妈说，小店以前谁都来。”马小满说，“只要买过东西，就不可能不存在。”

沈星河把两本本子并排，对着手电光。奇怪的事发生了。

户口簿第一页的空格里，慢慢浮出一个字。

何。

何字下面又出现一横、一竖，像有人隔着很远的水面写字。

栖。

何栖。

马小满屏住呼吸：“有了！”

下一秒，院子外面的路灯灭了一盏。

不是跳闸。

那盏灯的灯泡从里面黑下去，玻璃罩里像有一小撮火被谁吹走。

赵砚秋脸色变了：“每恢复一个名字，就会收回一盏借来的灯。”

“借来的？”林照野问。

沈岚低声说：“引魂灯的灯芯木，来自无声林。”

阿铃摸着自己的空白姓名牌：“所以它不是来抢东西。”

“它是来讨债。”沈星河说。

户口簿上第二个名字开始浮出。

刚出一笔，整条后院的灯都闪了一下。

马小满赶紧按住本子：“等等！要是名字全回来，县城会不会全黑？”

没人能回答。

窗外传来脚步声。

派出所值班室方向有人喊：“谁在后院？”

沈岚把手电一关。

黑暗里，户口簿上浮出的“何栖”两个字没有消失。它们发着极微弱的光，不像墨水，更像一小截从树心里剥出来的灯芯。

阿铃忽然抬头。

“他来了。”

院子中央，何栖站在熄灭的路灯下。

他看着那本户口簿，眼里第一次有了光。

“你们写了我的名字。”他说。

停了一下，他又问：

“那你们知道我吃过什么糖吗？”
