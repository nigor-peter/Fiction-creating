# 《雾岭灯火》后续写作思路

## 当前进度

第一章到第四章完成了第一阶段：周尧失踪、旧电影院线索、灯市街初入、两个周尧的谜题。

目前读者已经知道：

1. 灯市街真实存在，但更像两个世界之间的缝。
2. 地下青梧能借记忆、照片、声音和愿望模仿人。
3. 周尧至少出现了两个版本：白背心周尧和校服周尧。
4. 林成川、沈岚都知道旧案，但都在隐瞒。
5. 阿铃已经短暂出现，她像是知道规则的人。

## 第五章：灯戏唱本

核心任务：把民俗线正式接进主线。

建议开场：沈岚把沈星河带回家，母女第一次正面冲突。另一边，林照野发现相机渗出的蓝线和照片里周尧校服袖口吻合。

主要推进：

1. 孩子们需要弄懂“山下有城，城下有灯”的唱词。
2. 线索指向县文化馆研究员赵砚秋。
3. 赵砚秋表面温和，拒绝给他们看旧县志，只说“灯戏不是给活人看的”。
4. 顾南舟发现哥哥顾北川三年前曾借过同一本灯戏唱本。
5. 唱本缺了一页，缺页处正好写到“回头井”。
6. 阿铃再次出现，唱出缺失的一段。

结尾钩子：唱词显示，引魂灯每三十年必须熄灭一次；如果灯不灭，归人会越来越像活人。

## 第六章：实验井

核心任务：揭开现代实验线。

主要推进：

1. 沈星河追问父亲失踪真相。
2. 沈岚被迫承认十年前雾岭有一个“地磁异常观测项目”。
3. 林成川当年不是路过，而是项目的随队摄影。
4. 顾北川三年前可能不是误入，而是在追查旧实验资料。
5. 实验井与回头井不是同一口井，但两者在某些时刻会重叠。

结尾钩子：沈星河用收音机接入实验井旧频段，听见父亲说：“别让七月十五的灯亮起来。”

## 第七章：归人无声

核心任务：让诱惑扩大到全县。

主要推进：

1. 青梧县开始出现失踪者回来的传闻。
2. 周尧妈妈在巷口看见“周尧”，几乎认下他。
3. 马小满再次遇见外婆，这次外婆说出一个只有家里人才知道的秘密。
4. 顾南舟见到哥哥顾北川，哥哥劝他不要熄灯。
5. 阿铃告诉林照野：真正打开门的不是灯，是有人愿意相信影子是真的。

结尾钩子：全县停电预告提前，七月十五不再是十几天后，而是“今晚”。

## 第八章：雾岭灯火

核心任务：季终救人与关门。

主要推进：

1. 雾岭亮起千百盏灯，灯市街与现实老街重叠。
2. 两个周尧同时回到周尧妈妈面前。
3. 孩子们必须判断谁能救、谁不能认。
4. 林成川和沈岚进入成人线赎罪：他们提供旧实验路线和引魂灯位置，但最终选择由孩子完成。
5. 阿铃揭示自己不是普通孩子，她与最早的引魂灯祭祀有关。
6. 引魂灯被熄灭，裂隙暂时关闭。

结尾钩子：阿铃消失。照片里出现沈星河父亲，站在清代牌楼下，手里举着刚点燃的灯。

## 写作控制

每章保留一个明确道具：第五章唱本，第六章实验记录，第七章归人名单，第八章引魂灯。

每章推进一个成人秘密，但不要一次讲完。

两个周尧的判断要持续到季终，不要太早给答案。白背心周尧更像真实幸存者，校服周尧更像记忆模仿，但后续可以反转：真实周尧也可能被地下青梧污染，影子也可能学会保护别人。

阿铃不要解释太快。她每次出现只给一个规则、一句唱词或一个方向。
