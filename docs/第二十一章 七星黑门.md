# 第二十一章 七星黑门

开学第一天，青梧县小学所有指南针都坏了。

自然课老师把一只指南针放在讲台上，红色指针先是晃了两下，随后坚定地指向西边。

“这是北。”老师说。

沈星河看了一眼窗外。

西边是青梧水库。

她从文具盒里拿出自己修过的小指南针。指针同样转向西边。前排同学钥匙扣上的指南针、地理教室墙上的大罗盘，甚至老师用磁铁演示时悬在水面的缝衣针，都指向同一个方向。

马小满举手：“老师，有没有可能今天北边搬家了？”

全班笑起来。

老师没笑。

她盯着窗外，脸色有点白。

窗外操场上，十几个低年级孩子正同时转头看向水库。

当天晚上，第一个孩子梦游了。

他穿着背心和短裤，赤脚走出家门，一路往废弃水运码头去。父母追上时，他闭着眼，嘴里不停说：“船要开了。”

十二点以前，又有八个孩子走上老街。

他们来自不同年级，互不认识，走的却是同一条路。

林照野他们赶到水运码头时，黑门已经出现了。

它立在废弃候船室中央，门框漆黑，没有墙，也没有锁。门上画着七颗星，排成弯曲的勺形。第一颗星正发出微弱白光。

梦游的孩子们站在门前，齐声说：

“船要开了。”

周尧挡在最前面。

他是五个人里唯一听得懂门后声音的人。

“里面有人数数。”他说，“从一数到七。”

顾南舟问：“阿铃？”

“不是。”

林照野举起相机。

取景框里，黑门后是一片水。水面没有倒影，只有一艘木船。船没有船头，也没有船尾，七盏小灯沿船舷排列。

第二颗星亮了。

梦游的孩子同时往前一步。

马小满和顾南舟赶紧把人往后拖。沈星河取出那枚第二卷留下的星钉，靠近门缝。

星钉发热。

门后响起一声船铃。

叮。

第三颗星亮了。

“它不是在等七个孩子。”沈星河说，“是在等七个航标。”

门缝下漂出一只纸船。

纸已经湿透，却没有散。林照野捡起来，发现是用灯戏唱本复印页折的。正面写着一行歪斜的字：

不要来找我。

马小满叹气：“阿铃每次都这样。嘴上不让找，线索留得比老师布置作业还全。”

沈星河把纸船翻过来。

背面不是字，是一串数字和短线。

她脸色变了。

“第七号井频率的变式。”

“什么意思？”

“求救信号。”

第四颗星亮起。

黑门自行打开一条缝。门后没有风，却有水腥味和陌生的花香涌出来。梦游孩子中最小的女孩突然睁眼，指着门里说：“姐姐在船上。”

林照野问：“哪个姐姐？”

“没有脸的姐姐。”

沈星河将星钉插进门框第一颗星下方。

七星光芒同时停住。

梦游者像断线一样软倒，被赶来的家长抱走。码头重新安静，只剩黑门半开着。

林照野看向伙伴。

没人说去，也没人说不去。

第一卷他们为周尧进雾岭，第二卷为名字下沉城。这一次，门后只有一张写着“不要来”的纸船。

周尧先开口：“我去。”

顾南舟皱眉：“你不能每次都抢着失踪。”

“所以这次我跟紧。”

马小满把手电塞进书包：“我主要是去监督他。”

沈星河拔下星钉。

黑门上的七颗星依次亮起。

林照野最后看了一眼现实码头。远处老街灯火正常，学校开学通知还贴在墙上，水库只是安静的黑色水面。

他按下快门。

照片吐不出来。

相机里却响起阿铃的声音：

“上船以后，不要问哪边是回家。”

黑门完全打开。

五个人走了进去。

门在身后合上时，码头地面留下五道湿脚印。

第六道脚印跟在他们后面。

没有人看见。
