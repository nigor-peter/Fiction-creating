# 第二十五章 倒流河

进入倒流河后，伤口先消失了。

顾南舟手背那道在第一卷水泵房划出的疤慢慢合拢。周尧袖口缺掉的扣子重新缝回去。林照野相机皮套磨白的边缘恢复成新皮。

接着，话开始倒着说。

马小满刚说“我有点饿”，声音便从空气里退回他嘴里。他张嘴，却忘了自己想说什么。

“少说话。”沈星河在纸上写。

倒流河两岸不断出现过去的分岔口。

顾南舟看见三年前的顾北川站在文化馆门口。只要他跳下船，拦住哥哥借走唱本，就能阻止失踪。

他站在船边看了很久。

顾北川像感觉到什么，回头望向河上。

顾南舟握紧拳头，没有下船。

“他有权走那条路。”他把这句话写在船票上。

分岔口沉入河水。

沈星河看见十年前的县医院。

沈怀川正骑自行车离家，准备去第七号实验井。年轻的沈岚站在楼上晾衣服，三岁的沈星河在屋里睡觉。

她只要喊一声，父亲就会回头。

倒流河允许她改变这一刻。

沈星河站起来。

父亲的车铃声清晰得像就在耳边。

她握住船舷，手指发白。

最后，她把已经烧坏的收音机放到岸边。

不是阻止他去。

只是把未来那句“我听见了”留给过去的父亲。

河岸上的沈怀川停住自行车，像听见极远处有人回答。他没有回头，继续向雾岭骑去。

林照野看见父亲把海鸥相机递给年幼的自己。

如果他不接，第一卷的一切也许不会发生在他身上。他不会拍到周尧，不会进入灯市街，也不会一次次站在选择前。

他看着小时候的手接过相机。

没有阻止。

“相机不是错。”他说。

这一次，话没有倒流。

倒流河承认了这个选择。

江晚一直等自己的分岔口。

山洪那天终于出现。

白鹭镇庙会人声鼎沸。九岁的江潮提着纸风车，要和同伴去河边看龙舟。江晚在家门口洗衣服，嫌弟弟烦，挥手让他快走。

两小时后山洪暴发。

江晚跳下船。

“江晚！”沈星河喊。

她冲进分岔口，一把抓住江潮。

“今天不准去庙会。”

江潮被吓住：“为什么？”

“没有为什么。”

她把弟弟推进屋里，反锁门。

现实开始变化。

无向船上的木板长出白鹭镇码头才有的青苔。江晚运动衫上的校徽改变，山洪失踪名单从二十七人变成二十六人。

“你改了路。”沈星河说。

江晚回到船上，脸上第一次有了轻松。

“只改一个人。”

“你确定只是一个？”

河岸突然传来哭声。

因为江潮被锁在家里，原本和他同行的两个孩子没有在河边停下等他。他们提前到了下游，在山洪中失踪。

名单重新变成二十八人。

江晚脸上的轻松凝住。

“不可能。”

陆沉舟说：“改路不是挪一个人，是推整条河。”

江晚抓住船票：“那就再改。”

她想第二次跳下船，却被沈星河拦住。

“你会一直改，直到所有人都按你想的走。”

“那你让我怎么办？”江晚终于吼出来，“看着他死？”

“先问他去了哪里。”

倒流河尽头亮起一盏航标。

一个男孩站在灯下。

九岁，短裤，赤脚，手里拿着庙会纸风车。

江潮。

他没有被洪水困住。

他站在北斗渡第三座渡口，正在给迷路的船点灯。

江晚的船票从手中掉落。

“小潮？”

男孩抬头。

他看见姐姐，没有跑来。

只把航标举得更高。
