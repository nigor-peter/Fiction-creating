# 第二十四章 摆渡人

摆渡人卖的船票，是一段记忆。

他把小船靠过来，斗笠下面是一张被风雨磨得很深的脸。五十岁上下，左手少了一根小指，腰间挂着七枚铜牌。

“陆沉舟。”他自报姓名，“北斗渡摆渡人。”

马小满问：“船票多少钱？”

“不要钱。”

“那更贵。”马小满立刻警惕。

陆沉舟看了他一眼：“一人留下一段关于方向的记忆。”

“什么叫关于方向？”

“第一次离家，最后一次送别，或者某次明知不该去仍然走过去的路。”

船舷上出现五个凹槽。

顾南舟先把手放上去。

凹槽亮了一下。他想起哥哥失踪后，自己第一次独自走到雾岭山口。那段记忆被船取走，脑海里只剩事情发生过，却再也想不起当时山路的样子。

其他人依次买票。

林照野交出第一次跟父亲进暗房的路。

沈星河交出转学来青梧那天，从车站走到县医院的路线。

马小满交出外婆葬礼后自己绕远路回家的那个黄昏。

周尧交出从水泵房追向雾灯的碎石路。

江晚没有动。

陆沉舟说：“你已经付过。”

“我付了什么？”

“回白鹭镇的路。”

江晚脸色难看。

陆沉舟登上无向船，终于让中央罗盘停住。他解释七座渡口，每一座都保存未走之路。司南灯位于船最深处，持灯者可以改变航向。

“阿铃呢？”林照野问。

“她替你们关过两次门，身上有两盏灯留下的路。司南灯认她。”

“她有危险吗？”

“七条路线同时在叫她。”

“你为什么不救？”顾南舟问。

陆沉舟看着黑水：“摆渡人不替乘客选岸。”

马小满忍不住：“这话听着很有道理，其实就是不负责。”

陆沉舟竟然点头：“也可以这么说。”

他拿出六张船票。

每张票背面都印着一句话：请写明愿意舍弃的现实。

江晚第一个写。

她愿意舍弃白鹭镇山洪后的三天，换取二十七名失踪者存活。

沈星河看着她：“那三天里出生的人呢？”

“只有两个婴儿。我查过。”

“如果改路，他们可能不会出生。”

“二十七换二。”

“那两个孩子同意了吗？”

江晚抬头：“死去的二十七个人同意死了吗？”

两个人对视着。

她们都理性，也都勇敢。区别只是沈星河把每个具体的人视为不可代替，江晚认为不选择同样是在选择更多人受苦。

陆沉舟不评判。

“北斗渡只让代价被看见。”他说，“是否支付，由你们。”

林照野问：“司南灯能改变所有人的过去？”

“不能改变过去，只能让另一条本可能发生的路线，取代现在。”

“被取代的现在去哪？”

陆沉舟指向船下黑水。

“变成渡口。”

这意味着他们的世界不会彻底消失。

它会像未淹青梧一样，永远停在河岸，继续真实，却再也无法被选择了另一条路线的人回去。

船票在每个人手里发沉。

江晚把自己的票递给陆沉舟。

“带我去司南灯。”

“还差六座渡口。”

“我走。”

陆沉舟看向其他人：“你们呢？”

马小满把空票塞回他手里。

“我连暑假作业都不愿意替别人写，更别说替别人写人生。”

顾南舟说：“先找阿铃。”

沈星河点头。

陆沉舟敲响船铃。

第二颗星落到桅杆。

无向船前方出现一条逆流而上的河。水从低处流向高处，落下的雨滴从甲板升回天空。

“倒流河。”陆沉舟说，“在这里，你们会经过所有想收回的选择。”

船驶入逆流。

江晚手里的明信片自行展开。

纸上又多出一行江潮的字：

姐姐，别把我救成没做过选择的人。
