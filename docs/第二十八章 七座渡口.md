# 第二十八章 七座渡口

陆沉舟等了四十四年。

他摆渡过想救父母的人、想重来婚姻的人、想避开战争和洪水的人。每个人都问他哪条路最好，他从不回答。

不是因为中立。

是因为他希望有一天，有人替他证明当年选择女儿并没有错。

“你女儿后来呢？”马小满问。

陆沉舟看着熄灭的第七颗星。

“活到七十二岁，有三个孩子。她不知道陆家渡，也不知道我。”

“你后悔吗？”

“每天都后悔。每天也都不后悔。”

这才是选择真正的样子。

它不会因为结果好就失去代价，也不会因为代价大就证明当初全错。

沈星河把司南灯从桅杆上解下来。

“星图不是因为没人持灯才乱。”她说，“是七座渡口都被要求听一个人的方向。”

她拆开灯座。

里面不是一根灯芯，而是七枚铜钉。每枚钉上刻着一颗星。

“把它们送回七座渡口。”

陆沉舟皱眉：“航向会分裂。”

“本来就有七条路。”沈星河说，“为什么一定让一个人替所有人选？”

他们分开行动。

第一渡口，林照野守未淹青梧。他把现实照片放在码头，证明完整父亲和受伤父亲都是可能的人生，谁也无权替另一个覆盖。

第二渡口，顾南舟守倒流分岔。他重新看见哥哥失踪前的那天，却没有阻止。不是认可失去，而是尊重顾北川知道危险后仍做出的选择。

第三渡口，江晚与江潮一起守山洪航标。江晚撤回自己在倒流河做的改动。江潮仍然失踪，二十八人名单恢复为二十七。她哭着接受自己不能把弟弟爱成一个没有选择的人。

第四渡口，马小满面对一座所有人都平安、唯独无人记得彼此姓名的小镇。他把第二卷点名册钉在渡口，告诉那些人：活着不只是人数，还包括谁会在饭桌上叫你回来。

第五渡口，周尧守一条没有自己的青梧路线。他把寻人启事贴在航标上。

“我回来不是占了谁的位置。”他说，“我就是发生过。”

第六渡口，阿铃面对无数时代的灯路。每条路都要求她继续做送灯人。她把自己的姓名牌挂上去。

“我先是阿铃，才是送灯人。”

第七渡口属于陆沉舟。

他站在消失的陆家渡前，不敢落下最后一枚星钉。

村里有人做饭，有狗叫，女儿七岁，正站在桥上喊爹。只要他拔回当年的选择，村庄会恢复，女儿后来的一生和三个孩子则会消失。

马小满没有劝。

江晚也没有计算人数。

陆沉舟最终把星钉钉进村口碑。

不是恢复村庄。

只是让现实地图重新记下：这里曾有陆家渡，四十九户，一百八十六人。

“记录不是复活。”林照野说。

陆沉舟点头：“但也不是没有。”

七颗星重新亮起。

星图归位了一瞬。

就在这时，江晚的船票飞向司南灯。

她在第一座船上写下的愿望仍然有效：用现有白鹭镇换回山洪失踪者。

七颗星被愿望同时点亮。

无向船自动转舵。

江晚脸色变了：“不是我！”

陆沉舟说：“你没有撤票。”

船驶向北斗勺口指向的黑暗。

那里没有岸，没有星，也没有倒影。

无岸处。

能够让一条路线取代所有路线的地方。
