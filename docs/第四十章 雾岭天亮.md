# 第四十章 雾岭天亮

青梧县第一次在清晨看见雾岭完整的山脊。

雾没有一下散尽，而是像一条旧棉被，被谁从山肩上慢慢掀开。山路、林场、旧水泥厂、废弃电影院、县小学操场，一点点露在晨光里。

没有灯。

可天亮了。

县政府门口立起一块临时木牌：青梧县第零生产队补录处。

牌子写得不漂亮，墨迹还洇着。赵砚秋亲自搬桌子，沈岚整理旧病历，马莲拿着小卖部账本坐在旁边，谁来认一件旧物，她就问三遍：你确定？你还记得什么？有没有别人也知道？

这次没人嫌她啰嗦。

陈守根带着几个林场老人去西北沟立碑。碑文不是“失踪人员”，也不是“无名烈士”。

碑上刻的是：

青梧县第零生产队，曾在。

沈星河在林场边界最后一次收到父亲的声音。

那天午后，收音机自己亮了。她坐在树桩上，耳机线绕在手指间，听见熟悉的电流声从很远的地方来。

“星河。”

“我在。”

沈怀川的声音比以前清楚，却也更轻。

“灯标不用我守了。很多人开始记得，他们的声音比我一个人的声音稳。”

沈星河咬着嘴唇：“那你能回来吗？”

很长一段静。

“不能。”沈怀川说，“但我能走了。”

沈星河闭上眼。

她以为自己会哭得很厉害，可眼泪只是慢慢掉下来。原来真正听见告别时，人不一定会崩溃。人只是终于知道，该把哪一块疼放回原处。

“妈想你。”她说。

“我知道。”

“我也是。”

“我也知道。”

电流声里传来一声很轻的笑。

“好好长大，别总修别人留下的破东西。”

沈星河也笑了：“不保证。”

收音机红灯熄灭。

这一次，它没有再亮。

林照野拍下第一张真正的无声林照片。

照片里不是树，也不是雾，而是一群站在林场空地上的人。他们的脸终于显出来：有老人，有年轻人，有孩子，有人笑得拘谨，有人皱着眉像不习惯被拍。

照片背面，林照野没有写“失踪”。

他写：

曾在。

阿铃参加了期末考试。

她数学考得一塌糊涂，语文作文却写满了三页。题目是《我的名字》。老师看完后，把她叫到办公室，问她为什么写“名字不是别人给一次就够了，要每天有人叫，才会热”。

阿铃想了想，说：“因为凉了会熄。”

老师没听懂，但给了很高的分。

周尧的作文题目是《我被找过》。

他写：我以前以为被找回来就结束了。后来我知道，被找回来的人要学会找别人。因为不是每个人都有机会让别人知道自己不见了。

马小满看完，嘀咕：“你写这么好干什么，显得我写我家的狗很没深度。”

“你家没有狗。”顾南舟说。

“所以更有想象力。”

升初中前一天，五个人和阿铃又去了雾岭路口。

路边新竖了一块林区牌：无声林保护区，禁止采伐灯芯木。

牌子下面有人放了一小包桂花糖。

何栖没有出现。

但糖纸被风吹动时，林子里传来很轻的一下笑声。

林照野没有拍。

他只是把那声音记住。

傍晚回城时，他们在照相馆门缝里发现一封信。

信封没有邮票，没有寄信人，纸上也没有多余的话。

只有一句：

下一盏灯，不必再为死人点。

林照野把信递给阿铃。

阿铃看了很久，抬头望向雾岭。

山路上没有成排灯火，地下青梧的旧街没有浮现，北斗渡的星也没有亮。

只有县城一盏盏普通路灯，在黄昏里按时亮起来。

小卖部的灯，照相馆的灯，医院走廊的灯，学校门卫室的灯。

它们不神秘，也不古老。

可是每一盏灯下面，都有人能被叫出名字。

林成川在照相馆门口喊：“林照野，回家吃饭！”

马莲在街对面喊：“马小满，你再不回来我把剩菜全给你爸！”

顾怀声拄着拐杖站在剧团旧门口，喊顾南舟去练基本功。

沈岚推开医院办公室窗户，叫沈星河把收音机音量关小。

周尧妈妈从巷口探出头，喊儿子别又跑丢。

阿铃站在路灯下，等了一会儿。

马小满转身：“阿铃，走啊！”

她笑了。

“来了。”

雾岭山上，最后一缕雾散开。

没有锣鼓，没有唱腔，也没有谁从雾里回来。

只有天光落下来。

照见青梧县潮湿的街、旧墙、孩子们的影子，和一条终于不用回头也能走下去的路。
