# 第三十二章 没有声音的早读

第二天早读，青梧小学像被装进了玻璃瓶。

全校学生都站着背课文。嘴巴张开，喉咙动着，书页翻起一阵阵风，可教室里没有一个字。

老师的教鞭敲在讲台上，也没有声音。

起初大家以为是自己耳朵出了问题。直到隔壁班有个一年级孩子哭起来，眼泪挂了满脸，却连哭声都没发出来，整栋楼才真正乱了。

马小满把语文书举到脸前，夸张地念：“鹅鹅鹅，曲项向天歌。”

没人听见。

他脸上的笑一点点塌下去。

沈星河用铅笔在草稿纸上写：不是耳朵，是声音没出来。

顾南舟直接跳上窗台，用手势让操场上的体育老师看他们。体育老师站在旗杆旁，也在张嘴喊，脸憋得通红。

没有声音能穿过空气。

只有阿铃低着头，看自己的姓名牌。

昨天还裂着一条缝的白塑料牌上，“阿铃”两个字淡了很多，像被洗衣粉泡过。她用手指去按，指腹蹭下一点白粉。

林照野写：疼吗？

阿铃摇头。

她又点头。

沈星河从书包里拿出收音机。她没有开扬声器，而是拆下后盖，把耳机线接到天线口上。她动作很快，像在跟什么东西抢时间。

指针滑过县广播台、隔壁县交通台、短波杂讯。

依旧没有声音。

直到她拧到一个没有标记的频率。

耳机里忽然传来极轻的一下电流声。

沈星河猛地抬头。

“给我。”林照野无声地做口型。

沈星河把耳机递给他。

林照野戴上，只听见一截断断续续的呼吸。

然后，一个男人的声音从极远处挤出来。

“不要点灯。”

停顿。

“找树。”

声音断掉。

沈星河的脸白得吓人。

她把耳机抢回来，又调回刚才的频率。什么都没有了，只有一团像潮湿棉花的静。

“是你爸？”林照野在纸上写。

沈星河没有回答。

早读失声持续了整整七分钟。

声音恢复时，整栋楼像突然被倒进一锅水。读书声、哭声、桌椅声、老师的喊声同时炸开，连操场上的国旗绳都哗啦啦响。

班主任冲进来，第一句话是：“刚才谁在捣乱？”

没人能解释。

午休时，马小满拉着他们跑回小卖部。

“我妈说今天早上账本不对。”他说，“她明明记得有人赊过盐，翻开却空了。”

马小满家的小卖部还和以前一样挤：搪瓷盆、酱油瓶、玻璃柜里的糖、墙上挂着的塑料雨衣。马莲正坐在柜台后，把一本赊账本翻得哗哗响。

“奇了怪了。”她皱眉，“这几页是谁撕了？”

林照野看见本子没被撕。

纸还在，只是上面的字没了。

空白页中间留着油印似的凹痕。沈星河把纸斜着对光，看出几道被笔尖压过的线：盐二斤，火柴一盒，红糖半斤。

买东西的人名消失了。

马小满忽然从玻璃柜下面翻出一张旧糖纸。

糖纸被压得很平，上面写着一行铅笔字：欠小满家五分钱，等采木回来还。

落款的位置是空的。

“采木？”顾南舟问。

马莲听见这两个字，脸色变了：“谁说的？”

没人说话。

柜台上的老座钟“咔”地跳了一格。

阿铃胸前姓名牌上最后一点墨色也褪掉了。

白塑料牌空空荡荡，只剩别针扎出的一个小孔。

她摸着那个孔，小声说：“有人在叫我。”

“叫你什么？”林照野问。

阿铃张了张嘴。

小卖部里的声音忽然又被按掉。

货架最上层，一只多年没响过的铜铃轻轻晃动。

没有铃声。

只有窗外雾岭方向，一棵树被敲了三下。
