# 第十九章 送灯

赵砚萤知道所有关于妹妹的事。

她知道赵砚秋左眉有一道幼时摔伤的疤，知道兄妹俩曾在万灯巷九号偷听灯戏，知道山洪那晚赵砚秋答应回家取红雨鞋，却再也没能回来。

可她不知道自己后来去了哪里。

“你被救了。”阿铃走上戏台，“下游船家救了你。你长大，嫁人，有孩子，活到六十多岁。”

小女孩茫然地看着她。

赵砚秋猛地摇头：“不可能。”

阿铃从纸灯笼里取出一张旧纸。

那是下游县的户籍抄件：赵萤，原籍不详，洪灾获救。

名字少了一个“砚”字，生日、年岁和照片上的女孩都对得上。

“你等的妹妹，”阿铃说，“没有在等你。”

赵砚秋像一下子老了二十岁。

“那她为什么知道这些？”

“因为你知道。”

第二盏灯把他一生的记忆、愧疚和想象缝成一个女孩。她不是恶意的影子。她只是从赵砚秋不肯放下的那一夜里长出来。

赵砚萤低头看自己的手。

“那我是谁？”

这个问题和无名者问的一样。

赵砚秋张嘴，却答不出来。

阿铃把送灯纸灯笼放到戏台中央。

“不能直接灭第二盏灯。”她说，“它已经装了太多名字。灭掉，名字会一起散。”

沈星河问：“那怎么办？”

“把第一盏的归路和第二盏的旧名合在一起。让有名字的回记忆，让无名的走灯路。”

“谁来送？”林照野问。

阿铃没有回答。

所有人都明白了。

“不行。”马小满抱紧纸灯笼，“你上次已经掉下去一次。”

阿铃笑了一下：“所以我认得路。”

“回来呢？”

“送灯人不回头。”

马小满眼睛一下子红了：“这规矩是谁定的？”

阿铃想了想。

“以前的人。”

“以前的人也会定错。”马小满说。

阿铃愣住。

林照野看向两盏灯的线路。第一盏灯已灭，但灯芯保存在胶卷盒里；第二盏灯就在赵砚秋脚边。它们一盏管路，一盏管名。

“送灯人不该只有一个。”他说。

沈星河立刻明白：“把路线拆开。收音机线圈虽然烧了，扬声器还在，可以让现实的人一起叫名。”

马小满举起点名册：“我负责记。”

顾南舟握住木刀：“我走边开路。”

林照野拿起相机：“我把路拍下来。”

阿铃看着他们。

几百年来，她一直一个人提最后一盏灯。

第一次有人告诉她，规矩可以改。

顾北川走到顾南舟面前。

他的身体正在褪色。

“弟。”

顾南舟抬头。

“这次我不带你回去。”

“我知道。”

“你会去哪？”

顾北川看向阿铃手中的灯路。

“不知道。”

顾南舟点头。

他没有拥抱哥哥。两人之间隔着生死、记忆和三年没走完的路，任何触碰都可能变成认领。

顾南舟只按戏台规矩，向他抱拳行礼。

顾北川也抱拳。

像一出戏终于到了谢幕。

赵砚秋还跪在第二盏灯前。

小女孩拉住他的衣角。

“哥哥。”

她终于叫了。

赵砚秋浑身一震。

只要他回答，她就会占住赵砚萤的名字，那个真实活过一生的妹妹将从世上最后的记录里消失。

赵砚秋哭了很久。

最后，他轻轻掰开女孩的手。

“你不是砚萤。”

女孩脸上露出害怕。

赵砚秋继续说：“但你也不是假的。”

他取下自己的姓名牌，挂到女孩胸前。

“你是我记了太久的一个夜晚。”

女孩低头看木牌。

上面没有赵砚萤。

赵砚秋写了两个新字：雨灯。

“这是你的名字。”

女孩第一次有了影子。

第二盏灯剧烈摇晃。

沉城上方传来水库堤坝开裂的巨响。现实青梧与旧城开始重叠，万灯巷的门牌一块块浮上水面。

阿铃提起合成的送灯。

“来不及了。”

她看向四个孩子。

“一起走。”
